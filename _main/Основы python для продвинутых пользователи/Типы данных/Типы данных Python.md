# Типы данных Python

Встроенные типы данных Python — фундаментальные структуры для работы с данными.

## Классификация типов

| Категория | Типы | Изменяемость |
|-----------|------|--------------|
| **Числовые** | `int`, `float`, `complex` | Неизменяемые |
| **Последовательности** | `str`, `list`, `tuple`, `range` | `list` — изменяемый, остальные — нет |
| **Отображения** | `dict` | Изменяемый |
| **Множества** | `set`, `frozenset` | `set` — изменяемый, `frozenset` — нет |
| **Булев** | `bool` | Неизменяемый |
| **Байтовые** | `bytes`, `bytearray`, `memoryview` | `bytearray`, `memoryview` — изменяемые |

## Общие операции

### Для всех последовательностей
| Операция | Описание |
|----------|----------|
| `x in s` | Проверка вхождения |
| `s + t` | Конкатенация |
| `s * n` | Повторение |
| `s[i]` | Индексация |
| `s[i:j]` | Срез |
| `s[i:j:k]` | Срез с шагом |
| `len(s)` | Длина |
| `min(s)` | Минимальный элемент |
| `max(s)` | Максимальный элемент |
| `s.index(x)` | Индекс первого вхождения |
| `s.count(x)` | Количество вхождений |

### Проверка типов
```python
isinstance(obj, type)  # Проверка принадлежности к типу
type(obj)              # Получить тип объекта
```
