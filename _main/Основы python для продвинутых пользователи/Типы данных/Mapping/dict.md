# dict — Словари

**Категория:** [[Типы данных Python|Отображения]]

## Описание

Изменяемое отображение (mapping) ключ-значение. Ключи должны быть хешируемыми и уникальными.

Реализован как хеш-таблица с амортизированным O(1) для операций доступа.

## Создание

```python
# Литералы
d1 = {}
d2 = {'key1': 'value1', 'key2': 42}
d3 = {1: 'one', 2.0: 'two'}  # Ключи разных типов

# Конструктор
d4 = dict()                    # {}
d5 = dict([('a', 1), ('b', 2)])  # Из списка пар
d6 = dict(a=1, b=2)            # Из именованных аргументов
d7 = dict(zip(['a', 'b'], [1, 2]))  # Из двух списков

# Dict comprehension
squares = {x: x**2 for x in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# С условием
evens = {x: x**2 for x in range(10) if x % 2 == 0}
```

## Операции

| Операция | Синтаксис | Пример | Результат |
|----------|-----------|--------|-----------|
| Доступ | `d[key]` | `d['name']` | Значение или KeyError |
| Установка | `d[key] = value` | `d['age'] = 30` | Добавляет/обновляет |
| Удаление | `del d[key]` | `del d['age']` | Удаляет ключ |
| Вхождение | `key in d` | `'name' in d` | `True`/`False` |
| Длина | `len(d)` | `len(d)` | Количество пар |

## Методы

### Доступ к значениям

| Метод | Описание | Пример |
|-------|----------|--------|
| `get(key[, default])` | Получить значение (без ошибки) | `d.get('name', 'Unknown')` |
| `setdefault(key[, default])` | Получить или установить default | `d.setdefault('count', 0)` |

```python
d = {'a': 1, 'b': 2}

d.get('a')              # 1
d.get('c')              # None (нет KeyError)
d.get('c', 'default')   # 'default'

# setdefault — полезно для группировки
d = {}
for key in ['a', 'b', 'a', 'c', 'b']:
    d.setdefault(key, []).append(1)
# {'a': [1, 1], 'b': [1, 1], 'c': [1]}
```

### Обновление и слияние

| Метод | Описание | Пример |
|-------|----------|--------|
| `update(other)` | Обновить из other | `d1.update(d2)` |
| `d1 \| d2` | Объединение (Python 3.9+) | `d1 \| d2` |
| `d1 \|= d2` | Объединение на месте (Python 3.9+) | `d1 \|= d2` |

```python
d1 = {'a': 1, 'b': 2}
d2 = {'b': 20, 'c': 3}

d1.update(d2)           # {'a': 1, 'b': 20, 'c': 3}

# Python 3.9+
d3 = d1 | d2            # Новый словарь
d1 |= d2                # Обновление на месте
```

### Удаление

| Метод | Описание | Пример |
|-------|----------|--------|
| `pop(key[, default])` | Удалить и вернуть значение | `d.pop('a')` |
| `popitem()` | Удалить и вернуть последнюю пару | `d.popitem()` |
| `clear()` | Очистить словарь | `d.clear()` |

```python
d = {'a': 1, 'b': 2, 'c': 3}

d.pop('a')              # 1, d = {'b': 2, 'c': 3}
d.pop('x', 'default')   # 'default' (нет KeyError)
d.popitem()             # ('c', 3), d = {'a': 1, 'b': 2}
d.clear()               # {}
```

### Представления (Views)

| Метод | Описание | Пример |
|-------|----------|--------|
| `keys()` | View ключей | `d.keys()` |
| `values()` | View значений | `d.values()` |
| `items()` | View пар (ключ, значение) | `d.items()` |

```python
d = {'a': 1, 'b': 2, 'c': 3}

list(d.keys())          # ['a', 'b', 'c']
list(d.values())        # [1, 2, 3]
list(d.items())         # [('a', 1), ('b', 2), ('c', 3)]

# Итерация
for key in d:           # То же что d.keys()
    print(key)

for key, value in d.items():
    print(key, value)

# Views динамические — отражают изменения
keys = d.keys()
d['d'] = 4
list(keys)              # ['a', 'b', 'c', 'd']
```

### Копирование

| Метод | Описание | Пример |
|-------|----------|--------|
| `copy()` | Поверхностная копия | `d.copy()` |

```python
d1 = {'a': 1, 'b': {'x': 10}}
d2 = d1.copy()

d2['a'] = 99            # d1['a'] всё ещё 1
d2['b']['x'] = 99       # d1['b']['x'] тоже 99! (поверхностная копия)

# Для глубокой копии:
import copy
d3 = copy.deepcopy(d1)
```

### fromkeys

```python
# Создать словарь с ключами из iterable и одинаковым значением
dict.fromkeys(['a', 'b', 'c'], 0)
# {'a': 0, 'b': 0, 'c': 0}

# Осторожно с изменяемыми значениями!
dict.fromkeys(['a', 'b', 'c'], [])  # Все ключи ссылаются на один список!
```

## Итерация

```python
d = {'a': 1, 'b': 2, 'c': 3}

# По ключам
for key in d:
    print(key)

# По значениям
for value in d.values():
    print(value)

# По парам
for key, value in d.items():
    print(key, value)

# С фильтром
for key, value in {k: v for k, v in d.items() if v > 1}.items():
    print(key, value)
```

## Сортировка словаря

```python
d = {'b': 2, 'a': 1, 'c': 3}

# Сортировка по ключам
sorted(d.items())           # [('a', 1), ('b', 2), ('c', 3)]
dict(sorted(d.items()))     # {'a': 1, 'b': 2, 'c': 3}

# Сортировка по значениям
sorted(d.items(), key=lambda x: x[1])

# Обратная сортировка
sorted(d.items(), reverse=True)
```

## Особенности

- **Изменяемый тип**
- **Не хешируемый:** нельзя использовать как ключ словаря
- **Ключи должны быть хешируемыми:** строки, числа, кортежи (не списки!)
- **Сохраняет порядок вставки** (Python 3.7+)
- **O(1) операции:** доступ, вставка, удаление (амортизированно)

```python
# Ключи должны быть хешируемыми
d = {
    'string': 1,
    42: 2,
    (1, 2): 3,
    # [1, 2]: 4  # TypeError!
}

# Порядок вставки сохраняется
d = {'c': 3, 'a': 1, 'b': 2}
list(d.keys())    # ['c', 'a', 'b']
```

## Словари в Python 3.9+

```python
# Операторы объединения
d1 = {'a': 1, 'b': 2}
d2 = {'b': 20, 'c': 3}

d1 | d2             # {'a': 1, 'b': 20, 'c': 3} (d2 переопределяет)
d1 |= d2            # d1 обновляется на месте

# Разность ключей
d1.keys() - d2.keys()    # {'a'}
d1.keys() & d2.keys()    # {'b'}
d1.keys() ^ d2.keys()    # {'a', 'c'}
```
