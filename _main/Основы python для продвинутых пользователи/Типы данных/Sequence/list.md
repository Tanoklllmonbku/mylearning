# list — Списки

**Категория:** [[Типы данных Python|Последовательности]]

## Описание

Изменяемая последовательность для хранения коллекций элементов. Поддерживает элементы любых типов.

## Создание

```python
# Литералы
lst1 = []
lst2 = [1, 2, 3]
lst3 = [1, "two", 3.0, [4, 5]]  # Разные типы, вложенные списки

# Конструктор
lst4 = list()           # []
lst5 = list("hello")    # ['h', 'e', 'l', 'l', 'o']
lst6 = list(range(5))   # [0, 1, 2, 3, 4]
lst7 = list((1, 2, 3))  # [1, 2, 3] из кортежа

# List comprehension
squares = [x**2 for x in range(5)]           # [0, 1, 4, 9, 16]
evens = [x for x in range(10) if x % 2 == 0] # [0, 2, 4, 6, 8]
```

## Операции

| Операция | Синтаксис | Пример | Результат |
|----------|-----------|--------|-----------|
| Конкатенация | `lst1 + lst2` | `[1, 2] + [3, 4]` | `[1, 2, 3, 4]` |
| Повторение | `lst * n` | `[1, 2] * 3` | `[1, 2, 1, 2, 1, 2]` |
| Индексация | `lst[i]` | `[10, 20, 30][1]` | `20` |
| Отрицательный индекс | `lst[-i]` | `[10, 20, 30][-1]` | `30` |
| Срез | `lst[i:j]` | `[1,2,3,4][1:3]` | `[2, 3]` |
| Срез с шагом | `lst[i:j:k]` | `[1,2,3,4,5][::2]` | `[1, 3, 5]` |
| Разворот | `lst[::-1]` | `[1,2,3][::-1]` | `[3, 2, 1]` |
| Вхождение | `x in lst` | `2 in [1, 2, 3]` | `True` |
| Длина | `len(lst)` | `len([1, 2, 3])` | `3` |

## Методы

### Добавление элементов

| Метод | Описание | Пример |
|-------|----------|--------|
| `append(x)` | Добавить элемент в конец | `lst.append(4)` |
| `extend(iterable)` | Расширить список элементами iterable | `lst.extend([4, 5])` |
| `insert(i, x)` | Вставить элемент по индексу i | `lst.insert(1, 'x')` |

```python
lst = [1, 2, 3]
lst.append(4)           # [1, 2, 3, 4]
lst.extend([5, 6])      # [1, 2, 3, 4, 5, 6]
lst.insert(1, 'x')      # [1, 'x', 2, 3, 4, 5, 6]
```

### Удаление элементов

| Метод | Описание | Пример |
|-------|----------|--------|
| `remove(x)` | Удалить первое вхождение x (ValueError если нет) | `lst.remove(2)` |
| `pop([i])` | Удалить и вернуть элемент (последний по умолчанию) | `lst.pop()` |
| `clear()` | Очистить список | `lst.clear()` |

```python
lst = [1, 2, 3, 2, 4]
lst.remove(2)           # [1, 3, 2, 4] (первое вхождение)
lst.pop()               # [1, 3, 2], вернёт 4
lst.pop(0)              # [3, 2], вернёт 1
lst.clear()             # []
```

### Поиск и подсчёт

| Метод | Описание | Пример |
|-------|----------|--------|
| `index(x[, start, end])` | Индекс первого вхождения x | `lst.index(2)` |
| `count(x)` | Количество вхождений x | `lst.count(2)` |

```python
lst = [1, 2, 3, 2, 4]
lst.index(2)            # 1
lst.index(2, 2)         # 3 (начиная с индекса 2)
lst.count(2)            # 2
```

### Сортировка и изменение порядка

| Метод | Описание | Пример |
|-------|----------|--------|
| `sort(key=None, reverse=False)` | Сортировка на месте | `lst.sort()` |
| `reverse()` | Развернуть на месте | `lst.reverse()` |

```python
lst = [3, 1, 4, 1, 5]
lst.sort()              # [1, 1, 3, 4, 5]
lst.sort(reverse=True)  # [5, 4, 3, 1, 1]
lst.reverse()           # [1, 1, 3, 4, 5] → [5, 4, 3, 1, 1]

# Сортировка по ключу
words = ['apple', 'pie', 'a']
words.sort(key=len)     # ['a', 'pie', 'apple']
words.sort(key=str.lower)  # игнорирование регистра
```

### Копирование

| Метод | Описание | Пример |
|-------|----------|--------|
| `copy()` | Поверхностная копия | `lst.copy()` |

```python
lst1 = [1, 2, 3]
lst2 = lst1.copy()      # Новый список
lst2 = lst1[:]          # То же самое через срез

# Важно: поверхностная копия!
lst1 = [[1, 2], [3, 4]]
lst2 = lst1.copy()
lst2[0][0] = 99
lst1                    # [[99, 2], [3, 4]] — изменился и lst1!

# Для глубокой копии:
import copy
lst3 = copy.deepcopy(lst1)
```

## List comprehension

```python
# Базовый синтаксис
[x**2 for x in range(5)]                    # [0, 1, 4, 9, 16]

# С условием
[x for x in range(10) if x % 2 == 0]        # [0, 2, 4, 6, 8]

# Вложенные циклы
[(x, y) for x in range(3) for y in range(3)]  # [(0,0), (0,1), (0,2), (1,0), ...]

# С выражением-условием
[x if x > 0 else 0 for x in [-1, 2, -3, 4]]   # [0, 2, 0, 4]

# Вложенные списки
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
[row[0] for row in matrix]                  # [1, 4, 7]
```

## Распаковка

```python
# Базовая распаковка
a, b, c = [1, 2, 3]

# Звёздочка для остатка
first, *rest, last = [1, 2, 3, 4, 5]
# first=1, rest=[2,3,4], last=5

# Обмен значений
a, b = b, a

# Вложенная распаковка
(a, b), (c, d) = [[1, 2], [3, 4]]
```

## Особенности

- **Изменяемый тип:** методы изменяют список на месте
- **Не хешируемый:** нельзя использовать как ключ словаря
- **Динамический размер:** автоматически расширяется
- **Хранит ссылки:** элементы — ссылки на объекты

```python
# Производительность
# append() — O(1) амортизированное
# insert(0, x) — O(n) — сдвиг всех элементов
# pop() — O(1), pop(0) — O(n)
# x in lst — O(n)

# Для частых вставок в начало использовать collections.deque
from collections import deque
dq = deque([1, 2, 3])
dq.appendleft(0)    # O(1)
```
