# tuple — Кортежи

**Категория:** [[Типы данных Python|Последовательности]]

## Описание

Неизменяемая последовательность. Обычно используется для хранения разнородных данных (записей).

## Создание

```python
# Литералы
t1 = ()                  # Пустой кортеж
t2 = (1, 2, 3)           # Кортеж из 3 элементов
t3 = 1, 2, 3             # Скобки необязательны (упаковка)

# Кортеж из одного элемента (нужна запятая!)
t4 = (42,)               # (42,) — кортеж
t5 = 42,                 # (42,) — тоже кортеж
t6 = (42)                # 42 — просто число в скобках!

# Конструктор
t7 = tuple()             # ()
t8 = tuple("hello")      # ('h', 'e', 'l', 'l', 'o')
t9 = tuple([1, 2, 3])    # (1, 2, 3) из списка
```

## Операции

| Операция | Синтаксис | Пример | Результат |
|----------|-----------|--------|-----------|
| Конкатенация | `t1 + t2` | `(1, 2) + (3, 4)` | `(1, 2, 3, 4)` |
| Повторение | `t * n` | `(1, 2) * 3` | `(1, 2, 1, 2, 1, 2)` |
| Индексация | `t[i]` | `(10, 20, 30)[1]` | `20` |
| Отрицательный индекс | `t[-i]` | `(10, 20, 30)[-1]` | `30` |
| Срез | `t[i:j]` | `(1,2,3,4)[1:3]` | `(2, 3)` |
| Срез с шагом | `t[i:j:k]` | `(1,2,3,4,5)[::2]` | `(1, 3, 5)` |
| Вхождение | `x in t` | `2 in (1, 2, 3)` | `True` |
| Длина | `len(t)` | `len((1, 2, 3))` | `3` |

## Методы

Только методы чтения (кортеж неизменяем):

| Метод | Описание | Пример |
|-------|----------|--------|
| `count(x)` | Количество вхождений x | `(1,2,2,3).count(2)` → `2` |
| `index(x[, start, end])` | Индекс первого вхождения x | `(1,2,3,2).index(2)` → `1` |

```python
t = (1, 2, 3, 2, 4)
t.count(2)              # 2
t.index(2)              # 1
t.index(2, 2)           # 3 (начиная с индекса 2)
```

## Распаковка кортежей

```python
# Базовая распаковка
a, b, c = (1, 2, 3)

# Обмен значений
a, b = b, a

# Распаковка с звёздочкой
first, *middle, last = (1, 2, 3, 4, 5)
# first=1, middle=[2,3,4], last=5

# Вложенная распаковка
(a, b), (c, d) = ((1, 2), (3, 4))
```

## Кортежи как записи

```python
# Именованные кортежи для читаемости
from collections import namedtuple

Point = namedtuple('Point', ['x', 'y'])
p = Point(10, 20)
p.x                     # 10
p.y                     # 20

# Или с type hints (Python 3.6+)
from typing import NamedTuple

class Point(NamedTuple):
    x: int
    y: int

p = Point(10, 20)
```

## Кортежи как ключи

```python
# Кортежи хешируемы (если элементы хешируемы)
locations = {
    (40.7128, -74.0060): "New York",
    (51.5074, -0.1278): "London",
}

locations[(40.7128, -74.0060)]    # "New York"

# Списки нельзя использовать как ключи
# {[1, 2]: "value"}  # TypeError!
```

## Возврат нескольких значений

```python
def get_min_max(numbers):
    return min(numbers), max(numbers)

result = get_min_max([1, 2, 3, 4, 5])    # (1, 5)
min_val, max_val = get_min_max([1, 2, 3, 4, 5])
```

## Особенности

- **Неизменяемый тип:** нельзя изменить элементы после создания
- **Хешируемый:** можно использовать как ключ словаря (если элементы хешируемы)
- **Эффекнее списков:** меньше накладных расходов
- **Семантика:** используется для разнородных данных (записей)

```python
# Сравнение со списками
import sys

lst = [1, 2, 3, 4, 5]
tpl = (1, 2, 3, 4, 5)

sys.getsizeof(lst)    # Больше (списки динамические)
sys.getsizeof(tpl)    # Меньше (фиксированный размер)

# Производительность
# Доступ к элементам кортежа быстрее
# Кортежи можно использовать в качестве констант
```

## Когда использовать кортежи

| Сценарий | Тип |
|----------|-----|
| Коллекция однородных данных, нужно изменять | `list` |
| Запись с разнородными полями | `tuple` |
| Ключ словаря | `tuple` |
| Возврат нескольких значений из функции | `tuple` |
| Константы конфигурации | `tuple` |
