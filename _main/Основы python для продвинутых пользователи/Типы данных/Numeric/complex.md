# complex — Комплексные числа

**Категория:** [[Типы данных Python|Числовые типы]]

## Описание

Комплексные числа вида `a + bj`, где:
- `a` — действительная часть (real)
- `b` — мнимая часть (imag)
- `j` или `J` — мнимая единица (√-1)

## Создание

```python
# Литералы
z1 = 3 + 4j
z2 = -2.5 - 1.5j
z3 = 0j

# Из чисел
z4 = complex(3, 4)      # (3+4j)
z5 = complex(5)         # (5+0j)
z6 = complex("3+4j")    # (3+4j)

# Пустое значение
z7 = complex()          # 0j
```

## Атрибуты

```python
z = 3 + 4j

z.real    # 3.0  (действительная часть, всегда float)
z.imag    # 4.0  (мнимая часть, всегда float)
```

## Арифметические операции

| Операция | Синтаксис | Пример | Результат |
|----------|-----------|--------|-----------|
| Сложение | `z1 + z2` | `(1+2j) + (3+4j)` | `(4+6j)` |
| Вычитание | `z1 - z2` | `(5+3j) - (2+1j)` | `(3+2j)` |
| Умножение | `z1 * z2` | `(1+2j) * (3+4j)` | `(-5+10j)` |
| Деление | `z1 / z2` | `(1+2j) / (3+4j)` | `(0.44+0.08j)` |
| Модуль | `abs(z)` | `abs(3+4j)` | `5.0` |
| Сопряжённое | `z.conjugate()` | `(3+4j).conjugate()` | `(3-4j)` |

**Не поддерживаются:** `//`, `%`, битовые операции

## Методы

### `complex.conjugate()`
Возвращает сопряжённое число (меняет знак мнимой части).

```python
z = 3 + 4j
z.conjugate()    # (3-4j)

# Оператор сопряжения
z.conjugate() == z.conj()    # True (есть алиас)
```

### `complex.from_number(x)` (Python 3.14+)
Создаёт комплексное число из другого числа.

```python
complex.from_number(5)       # (5+0j)
complex.from_number(3.14)    # (3.14+0j)
```

## Модуль cmath

Для работы с комплексными числами используется модуль `cmath`:

```python
import cmath

z = 3 + 4j

cmath.phase(z)           # 0.927... (аргумент/угол в радианах)
cmath.polar(z)           # (5.0, 0.927...) (модуль и аргумент)
cmath.rect(5, 0.927)     # (3+4j) (из полярных координат)

# Тригонометрические функции
cmath.sin(z)
cmath.cos(z)
cmath.tan(z)

# Гиперболические функции
cmath.sinh(z)
cmath.cosh(z)
cmath.tanh(z)

# Экспонента и логарифмы
cmath.exp(z)
cmath.log(z)
cmath.log10(z)
cmath.sqrt(z)
```

## Полярные координаты

```python
import cmath

z = 3 + 4j

# Декартовы → Полярные
r, phi = cmath.polar(z)    # r = 5.0, phi = 0.927...

# Полярные → Декартовы
z = cmath.rect(r, phi)     # (3+4j)
```

## Примеры использования

```python
# Поворот на угол
def rotate(z, angle):
    import cmath
    return z * cmath.rect(1, angle)

z = 1 + 0j
rotate(z, cmath.pi / 2)    # (~0+1j) — поворот на 90°

# Импеданс в электротехнике
Z = 100 + 50j  # Активное + реактивное сопротивление
abs(Z)         # 111.8 Ом — полный импеданс
```

## Особенности

- **Неизменяемый тип**
- **Хешируемый**
- **Мнимая единица:** `j` или `J` (инженерное обозначение)
- **Обе части всегда float**

```python
# Проверка на комплексное число
isinstance(3+4j, complex)    # True
isinstance(5, complex)       # False

# Преобразование типов
complex(5)         # (5+0j)
int(3+0j)          # TypeError! (нельзя напрямую)
```
