# int — Целые числа

**Категория:** [[Типы данных Python|Числовые типы]]

## Описание

Целочисленный тип с **неограниченной точностью**. Размер ограничен только доступной памятью.

## Создание

```python
# Литералы
x = 42
y = -17
z = 0

# Из других типов
a = int(3.14)      # 3 (отбрасывает дробную часть)
b = int("123")     # 123
c = int("FF", 16)  # 255 (шестнадцатеричное)
d = int(0b1010)    # 10 (двоичное)

# Пустое значение
e = int()          # 0
```

## Системы счисления

| Система | Префикс | Пример | Значение |
|---------|---------|--------|----------|
| Десятичная | нет | `42` | 42 |
| Двоичная | `0b` | `0b101010` | 42 |
| Восьмеричная | `0o` | `0o52` | 42 |
| Шестнадцатеричная | `0x` | `0x2A` | 42 |

## Арифметические операции

| Операция | Синтаксис | Пример | Результат |
|----------|-----------|--------|-----------|
| Сложение | `x + y` | `5 + 3` | `8` |
| Вычитание | `x - y` | `5 - 3` | `2` |
| Умножение | `x * y` | `5 * 3` | `15` |
| Деление | `x / y` | `5 / 3` | `1.666...` (float) |
| Целочисленное деление | `x // y` | `5 // 3` | `1` |
| Остаток | `x % y` | `5 % 3` | `2` |
| Возведение в степень | `x ** y` | `5 ** 3` | `125` |
| Модуль | `abs(x)` | `abs(-5)` | `5` |
| Пара (частное, остаток) | `divmod(x, y)` | `divmod(5, 3)` | `(1, 2)` |

## Битовые операции

| Операция | Синтаксис | Пример | Результат |
|----------|-----------|--------|-----------|
| И | `x & y` | `5 & 3` | `1` |
| ИЛИ | `x \| y` | `5 \| 3` | `7` |
| Исключающее ИЛИ | `x ^ y` | `5 ^ 3` | `6` |
| Сдвиг влево | `x << n` | `5 << 1` | `10` |
| Сдвиг вправо | `x >> n` | `5 >> 1` | `2` |
| Инверсия | `~x` | `~5` | `-6` |

## Методы

### `int.bit_length()`
Возвращает количество бит, необходимых для представления числа в двоичной системе (без знака и ведущих нулей).

```python
(42).bit_length()      # 6  (101010)
(-42).bit_length()     # 6
(0).bit_length()       # 0
(255).bit_length()     # 8
```

### `int.bit_count()`
Возвращает количество единиц в двоичном представлении (Python 3.10+).

```python
(42).bit_count()       # 3  (101010 → три единицы)
(255).bit_count()      # 8  (11111111)
```

### `int.to_bytes(length, byteorder, signed=False)`
Преобразует int в байты заданной длины.

```python
(1024).to_bytes(2, 'big')           # b'\x04\x00'
(1024).to_bytes(2, 'little')        # b'\x00\x04'
(-1024).to_bytes(2, 'big', signed=True)  # b'\xfc\x00'
```

### `int.from_bytes(bytes, byteorder, signed=False)`
Создаёт int из байтов.

```python
int.from_bytes(b'\x04\x00', 'big')           # 1024
int.from_bytes(b'\x00\x04', 'little')        # 1024
int.from_bytes(b'\xfc\x00', 'big', signed=True)  # -1024
```

### `int.as_integer_ratio()`
Возвращает кортеж `(числитель, знаменатель)`.

```python
(5).as_integer_ratio()   # (5, 1)
(-3).as_integer_ratio()  # (-3, 1)
```

## Сравнение

| Операция | Синтаксис | Пример |
|----------|-----------|--------|
| Равно | `x == y` | `5 == 5` → `True` |
| Не равно | `x != y` | `5 != 3` → `True` |
| Больше | `x > y` | `5 > 3` → `True` |
| Меньше | `x < y` | `3 < 5` → `True` |
| Больше или равно | `x >= y` | `5 >= 5` → `True` |
| Меньше или равно | `x <= y` | `3 <= 5` → `True` |

## Особенности

- **Неограниченная точность:** `10**1000` работает без переполнения
- **Неизменяемый тип:** операции создают новые объекты
- **Хешируемый:** можно использовать как ключ словаря

```python
# Пример большого числа
factorial_100 = 1
for i in range(1, 101):
    factorial_100 *= i
# 93326215443944152681699238856266700490715968264381621468592963895217599993229915608941463976156518286253697920827223758251185210916864000000000000000000000000
```
