# float — Числа с плавающей точкой

**Категория:** [[Типы данных Python|Числовые типы]]

## Описание

Вещественные числа двойной точности (аналог `double` в C). Реализуют стандарт IEEE 754.

**Точность:** ~15-17 значащих десятичных цифр

## Создание

```python
# Литералы
x = 3.14
y = -0.001
z = 0.0

# Научная нотация
a = 6.022e23     # 6.022 × 10^23
b = 1.6e-19      # 1.6 × 10^-19

# Из других типов
c = float(42)        # 42.0
d = float("3.14")    # 3.14
e = float("inf")     # бесконечность
f = float("nan")     # Not a Number

# Пустое значение
g = float()          # 0.0
```

## Специальные значения

| Значение | Создание | Описание |
|----------|----------|----------|
| Бесконечность | `float('inf')` | Положительная бесконечность |
| Отрицательная бесконечность | `float('-inf')` | Отрицательная бесконечность |
| NaN | `float('nan')` | Not a Number (не число) |

```python
import math
math.isinf(float('inf'))    # True
math.isnan(float('nan'))    # True
```

## Арифметические операции

| Операция | Синтаксис | Пример | Результат |
|----------|-----------|--------|-----------|
| Сложение | `x + y` | `5.5 + 2.3` | `7.8` |
| Вычитание | `x - y` | `5.5 - 2.3` | `3.2` |
| Умножение | `x * y` | `5.5 * 2.0` | `11.0` |
| Деление | `x / y` | `5.5 / 2.0` | `2.75` |
| Целочисленное деление | `x // y` | `5.5 // 2.0` | `2.0` |
| Остаток | `x % y` | `5.5 % 2.0` | `1.5` |
| Степень | `x ** y` | `2.0 ** 3.0` | `8.0` |

## Методы

### `float.as_integer_ratio()`
Возвращает точное представление числа как дробь `(числитель, знаменатель)`.

```python
(0.5).as_integer_ratio()    # (1, 2)
(2.5).as_integer_ratio()    # (5, 2)
(0.1).as_integer_ratio()    # (3602879701896397, 36028797018963968)
```

### `float.is_integer()`
Проверяет, является ли число целым.

```python
(5.0).is_integer()    # True
(5.5).is_integer()    # False
```

### `float.hex()`
Возвращает шестнадцатеричное представление.

```python
(3.14).hex()    # '0x1.91eb851eb851fp+1'
```

### `float.fromhex(s)`
Создаёт float из шестнадцатеричной строки.

```python
float.fromhex('0x1.91eb851eb851fp+1')    # 3.14
```

## Функции модуля math

```python
import math

math.trunc(3.7)      # 3   (отбрасывание дробной части)
math.floor(3.7)      # 4   (округление вниз)
math.ceil(3.2)       # 4   (округление вверх)
round(3.5)           # 4   (банковское округление)
round(3.14159, 2)    # 3.14 (округление до 2 знаков)
```

## Проблемы точности

```python
# Классическая проблема
0.1 + 0.2 == 0.3    # False!
0.1 + 0.2           # 0.30000000000000004

# Решение: использовать Decimal для финансовых расчётов
from decimal import Decimal
Decimal('0.1') + Decimal('0.2') == Decimal('0.3')  # True
```

## Сравнение

```python
# Прямое сравнение может быть неточным
x = 0.1 + 0.2
x == 0.3            # False

# Правильный способ: с допуском
import math
math.isclose(x, 0.3, rel_tol=1e-9)    # True
```

## Особенности

- **Неизменяемый тип**
- **Хешируемый** (кроме NaN)
- **Не точный:** не подходит для финансовых расчётов
- **Диапазон:** примерно от `1e-308` до `1e308`

```python
# Границы
import sys
sys.float_info.max      # ~1.8e308
sys.float_info.min      # ~2.2e-308
sys.float_info.epsilon  # ~2.2e-16 (машинный эпсилон)
```
