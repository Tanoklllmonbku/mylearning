# Комментарии C

Комментарии — текст исходного кода, игнорируемый компилятором. Используются для документирования и отладки.

## Типы комментариев

### 1. Блочные (многострочные)

```c
/* Это блочный комментарий */

/*
 * Многострочный комментарий
 * в стиле C
 */

int x = 5; /* комментарий после кода */
```

- Начинаются с `/*`, заканчиваются `*/`
- Могут охватывать несколько строк
- **Не вкладываются** друг в друга

### 2. Строчные (C99 и выше)

```c
// Строчный комментарий до конца строки
int y = 10; // комментарий после кода
```

- Начинаются с `//`
- Действуют до конца строки
- Появились в стандарте C99 (из C++)

## Стандарты

| Тип | Стандарт | Примечание |
|-----|----------|------------|
| `/* */` | C89/C90 | Базовый синтаксис |
| `//` | C99 | Заимствовано из C++ |

## Важные замечания

- Комментарии удаляются на этапе **лексического анализа**
- Вложенные блочные комментарии — **неопределённое поведение**:
  ```c
  /* внешний /* вложенный */ ошибка */  // НЕ работает!
  ```
- Можно использовать для **временного отключения кода**:
  ```c
  // old_code();
  new_code();
  ```
