# continue

**Категория:** Управление потоком

## Назначение

Пропуск остатка текущей итерации цикла и переход к следующей.

## Пример

```c
// Пропуск нечётных чисел
for (int i = 0; i < 10; i++) {
    if (i % 2 != 0) {
        continue;  // переход к следующей итерации
    }
    printf("%d ", i);  // выведет: 0 2 4 6 8
}
```

## Отличие от break

- `continue` — переход к **следующей итерации**
- `break` — **выход из цикла** полностью
