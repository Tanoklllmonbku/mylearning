# if

**Категория:** Управление потоком

## Назначение

Условное выполнение кода.

## Синтаксис

```c
if (condition) {
    // выполняется, если condition != 0
}
```

## Пример

```c
if (x > 0) {
    printf("Positive");
} else if (x < 0) {
    printf("Negative");
} else {
    printf("Zero");
}
```
