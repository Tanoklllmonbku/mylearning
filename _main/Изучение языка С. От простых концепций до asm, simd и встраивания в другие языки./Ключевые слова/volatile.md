# volatile

**Категория:** Квалификатор типа

## Назначение

Запрещает компилятору оптимизировать обращения к переменной. Вызвано потребностью в неизменяемости методов работы с переменной для одновременного общения данными между кодом и внешних средств (Системные прерывания, embedded, сигнальные переменные(Не заменяет мьютексы\атомарность!)).

## Применение

### 1. Встраиваемые системы (MMIO)

```c
volatile uint32_t* const PORT = (uint32_t*)0x4000;
*PORT = 1;  // обязательная запись в порт
```

### 2. Многопоточность

```c
volatile int flag = 0;

// Поток 1
flag = 1;

// Поток 2
while (flag == 0) { }
```

### 3. Сигналы

```c
volatile sig_atomic_t got_signal = 0;
```

## Важно

- **Не гарантирует атомарность**
