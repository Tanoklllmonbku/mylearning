# double

**Категория:** Тип данных

## Назначение

Вещественный тип двойной точности (64 бита, IEEE 754).

## Характеристики

| Параметр | Значение |
|----------|----------|
| Размер | 8 байт (64 бита) |
| Точность | ~15-17 десятичных знаков |
| Диапазон | ±1.7×10³⁰⁸ |

## Пример

```c
double pi = 3.14159265358979;
double large = 1.23e100;
double small = 4.56e-50;
```
