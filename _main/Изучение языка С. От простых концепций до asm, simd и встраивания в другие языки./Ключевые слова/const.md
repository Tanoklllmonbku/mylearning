# const

**Категория:** Квалификатор типа

## Назначение

Объявляет объект **только для чтения**. Попытка изменения вызывает ошибку компиляции.

## Пример

```c
const int MAX = 100;    // константа
MAX = 200;              // ОШИБКА!

const char* str = "text";  // указатель на константные данные
char* const ptr = buf;     // константный указатель
const char* const cp = s;  // константный указатель на константные данные
```

## Применение

- Константы и параметры только для чтения
- Оптимизация компилятора
- Защита от случайных изменений
