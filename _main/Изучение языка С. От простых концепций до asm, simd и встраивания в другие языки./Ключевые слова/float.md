# float

**Категория:** Тип данных

## Назначение

Вещественный тип одинарной точности (32 бита, IEEE 754).

## Характеристики

| Параметр | Значение |
|----------|----------|
| Размер | 4 байта (32 бита) |
| Точность | ~6-9 десятичных знаков |
| Диапазон | ±3.4×10³⁸ |

## Пример

```c
float x = 3.14f;           // суффикс f обязателен
float y = 2.5e-3f;
```
