# static

**Категория:** Класс хранения

## Назначение

Изменяет время жизни и область видимости переменных и функций.

## Применение

### 1. Локальные переменные

Сохраняют значение между вызовами функции.

```c
void counter() {
    static int count = 0;
    count++;
    printf("%d ", count);
}

counter();  // 1
counter();  // 2
```

### 2. Глобальные переменные и функции

Ограничивают видимость **текущим файлом**.

```c
static int global = 42;  // виден только в этом файле
static void helper() { }
```

## Пример из практики

Сравнение `static` и `auto` в [[Auto vs Static]]:

```c
void count() {
    static int x = 0;  // сохраняется между вызовами
    auto int y = 0;    // сбрасывается при каждом вызове

    x++;  // 1, 2, 3...
    y++;  // всегда 1

    printf("%d, %d\n", x, y);  // 1,1  2,1  3,1
}

int main() {
    for (int i = 0; i < 3; i++) {
        count();  // выводит: 1,1  2,1  3,1
    }
    return 0;
}
```

**Вывод:**
- `static int x` — инициализируется один раз, сохраняет значение (1→2→3)
- `auto int y` — инициализируется заново при каждом вызове (всегда 1)

## Сравнение с thread_local

| Аспект | `static` | `thread_local` |
|--------|----------|----------------|
| **Копий** | 1 на программу | 1 на поток |
| **Инициализация** | При загрузке | При первом входе потока |
| **Время жизни** | Вся программа | До завершения потока |
| **Пример** | Счётчик вызовов | `errno`, буфер на поток |

```c
static int counter = 0;           // общий для всех потоков
thread_local int thread_counter = 0;  // уникальный для каждого потока
```
