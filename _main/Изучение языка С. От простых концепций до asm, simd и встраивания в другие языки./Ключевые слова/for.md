# for

**Категория:** Управление потоком

## Назначение

Цикл со счётчиком.

## Синтаксис

```c
for (инициализация; условие; инкремент) {
    // тело цикла
}
```

## Пример

```c
for (int i = 0; i < 10; i++) {
    printf("%d ", i);
}

// Бесконечный цикл
for (;;) {
    if (done) break;
}
```

## Пример из практики

Цикл `for` в [[Калькулятор]] для парсинга токенов:

```c
// Обработка токенов в выражении
for (int i = 0; i < token_count; i++) {
    if (tokens[i].type == OPERATOR) {
        // обработка оператора
    }
}

// Вычисление скобок (с самых глубоких)
for (int depth = max_depth; depth > 0; depth--) {
    // вычисление выражений в скобках
}
```
