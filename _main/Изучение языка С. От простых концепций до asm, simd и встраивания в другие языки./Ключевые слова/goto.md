# goto

**Категория:** Управление потоком

## Назначение

Безусловный переход к метке в пределах **той же функции**.

## Пример

```c
// Выход из вложенных циклов
for (int i = 0; i < 10; i++) {
    for (int j = 0; j < 10; j++) {
        if (error) {
            goto cleanup;
        }
    }
}

cleanup:
    printf("Error occurred\n");
```

## Предупреждение

- **Не рекомендуется** (нарушает структурированность)
- Допустим для обработки ошибок
