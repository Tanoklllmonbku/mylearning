# else

**Категория:** Управление потоком

## Назначение

Альтернативная ветка в условном операторе `if`.

## Синтаксис

```c
if (condition) {
    // если истинно
} else {
    // если ложно
}
```

## Пример

```c
if (x > 0) {
    printf("Positive");
} else {
    printf("Non-positive");
}
```
