# inline

**Категория:** Спецификатор функции (C99)

## Назначение

Подсказка компилятору подставить тело функции **вместо вызова**. Увеличение размера бинарника, ниже задержка вызова (Только для High-RPS функций)

## Пример

```c
inline int max(int a, int b) {
    return a > b ? a : b;
}

int x = max(5, 10);
```

## Правила

- Компилятор **может проигнорировать** inline
- В заголовочных файлах требует `static inline`
