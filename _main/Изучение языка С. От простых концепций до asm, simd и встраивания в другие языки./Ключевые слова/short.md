# short

**Категория:** Модификатор типа

## Назначение

Целочисленный тип уменьшенного размера (минимум 2 байта).

## Характеристики

| Параметр | Значение |
|----------|---------|
| Размер | ≥ 2 байта (16 бит) |
| Диапазон (signed) | -32768 ... 32767 |

## Пример

```c
short s = 100;
unsigned short us = 60000U;
```
