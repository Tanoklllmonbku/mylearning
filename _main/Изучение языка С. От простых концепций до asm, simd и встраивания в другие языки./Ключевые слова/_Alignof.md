# _Alignof

**Категория:** Оператор (C11)

## Назначение

Получение требования выравнивания типа в байтах.

## Синтаксис

```c
_Alignof(тип)
```

## Пример

```c
printf("%zu", _Alignof(int));        // 4 (типично)
printf("%zu", _Alignof(double));     // 8
printf("%zu", _Alignof(max_align_t)); // 16 или 32
```

## Применение

- Отладка структур данных
- Оптимизация памяти
- SIMD и низкоуровневое программирование
