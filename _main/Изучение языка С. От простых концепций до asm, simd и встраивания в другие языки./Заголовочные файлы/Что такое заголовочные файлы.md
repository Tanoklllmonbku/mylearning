# Что такое заголовочные файлы

**Заголовочные файлы** (`.h`) — файлы с объявлениями функций, типов, макросов и внешних переменных.

## Назначение

1. **Объявления** — прототипы функций, typedef, extern переменные
2. **Макросы** — константы, функционоподобные макросы
3. **Переиспользование** — подключение в нескольких файлах

## Пример

```c
// mylib.h
#ifndef MYLIB_H
#define MYLIB_H

void my_function(int x);
int add(int a, int b);

#define MAX_SIZE 100

#endif
```

## Подключение

```c
// main.c
#include "mylib.h"
#include <stdio.h>

int main() {
    my_function(42);
    return 0;
}
```

## Типы заголовков

| Форма | Пример | Где ищет |
|-------|--------|----------|
| `"file.h"` | `#include "mylib.h"` | Текущая директория → системные |
| `<file.h>` | `#include <stdio.h>` | Системные директории |
