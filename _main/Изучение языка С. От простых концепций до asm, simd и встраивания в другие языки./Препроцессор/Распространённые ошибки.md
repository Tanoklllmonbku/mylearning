# Распространённые ошибки препроцессора

## 1. Отсутствие скобок

```c
// ОШИБКА
#define MUL(a, b) a * b
MUL(1 + 2, 3)  // 1 + 2 * 3 = 7

// ПРАВИЛЬНО
#define MUL(a, b) ((a) * (b))
```

## 2. Точка с запятой в определении

```c
// ОШИБКА
#define MAX 100;
int x = MAX;  // int x = 100;; — синтаксическая ошибка

// ПРАВИЛЬНО
#define MAX 100
```

## 3. Побочные эффекты в аргументах

```c
// ОШИБКА
#define SQUARE(x) ((x) * (x))
SQUARE(i++)  // i увеличивается дважды!

// ПРАВИЛЬНО: inline-функция
inline int square(int x) { return x * x; }
```

## 4. Missing do-while(0)

```c
// ОШИБКА
#define INIT() \
    x = 0; \
    y = 0

if (cond)
    INIT();  // только x = 0; в if!
else { }

// ПРАВИЛЬНО
#define INIT() \
    do { x = 0; y = 0; } while (0)
```

## 5. Конфликт имён

```c
// ОШИБКА
#define bool int
#define true 1
#define false 0

#include <stdbool.h>  // конфликт!

// ПРАВИЛЬНО
#define MY_BOOL int
#define MY_TRUE 1
#define MY_FALSE 0
```

## 6. Утечка макросов в заголовках

```c
// header.h — ОШИБКА
#define MAX 100
void process(int size);

// user.c
#include "header.h"
#undef MAX  // другие файлы сломаются!

// ПРАВИЛЬНО
#ifndef MAX
#define MAX 100
#endif
```

## 7. Бесконечная рекурсия

```c
// ОШИБКА
#define FOO BAR
#define BAR FOO
FOO  // зацикливание (препроцессор остановит)

// ПРАВИЛЬНО
#define FOO 42
```

## 8. Stringify без раскрытия

```c
// ОШИБКА
#define VALUE 42
#define STRINGIFY(x) #x
STRINGIFY(VALUE)  // "VALUE", а не "42"

// ПРАВИЛЬНО
#define TOSTRING(x) STRINGIFY(x)
TOSTRING(VALUE)  // "42"
```

## 9. Token pasting без раскрытия

```c
// ОШИБКА
#define X foo
#define Y bar
#define CONCAT(a, b) a##b
CONCAT(X, Y)  // XY

// ПРАВИЛЬНО
#define XCONCAT(a, b) CONCAT(a, b)
XCONCAT(X, Y)  // foobar
```

## 10. Forget Include Guards

```c
// ОШИБКА
// header.h без guards
struct Data { };

// main.c
#include "header.h"
#include "other.h"  // который включает header.h
// ошибка: повторное определение!

// ПРАВИЛЬНО
#ifndef HEADER_H
#define HEADER_H
struct Data { };
#endif
```
