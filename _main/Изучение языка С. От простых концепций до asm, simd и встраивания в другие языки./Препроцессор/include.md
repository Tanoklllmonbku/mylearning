# #include

**Категория:** Директива включения

## Назначение

Вставляет содержимое другого файла в текущее место.

## Синтаксис

```c
#include <файл>     // системные заголовки
#include "файл"     // пользовательские заголовки
```

## Различия

| Форма | Где ищет |
|-------|----------|
| `<файл>` | Системные директории компилятора |
| `"файл"` | Текущая директория, затем системные |

## Пример

```c
#include <stdio.h>      // printf, scanf
#include <stdlib.h>     // malloc, free
#include "myheader.h"   // свой заголовок
#include "../utils.h"   // относительный путь
```

## Что происходит

```c
// До препроцессора
#include <stdio.h>
int main() { printf("Hi"); }

// После препроцессора
// ... содержимое stdio.h ...
int main() { printf("Hi"); }
```

## Вложенные include

```c
// a.h
#include "b.h"
#define A 1

// main.c
#include "a.h"  // включает b.h автоматически
```
