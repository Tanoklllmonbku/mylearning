# Условная компиляция

**Условная компиляция** — включение/исключение кода в зависимости от условий.

## Директивы

- `#if` — условие с выражением
- `#ifdef` — если макрос определён
- `#ifndef` — если макрос не определён
- `#elif` — иначе-если
- `#else` — иначе
- `#endif` — конец блока

## Примеры

### Базовое использование

```c
#define DEBUG

#ifdef DEBUG
    printf("Debug info\n");
#endif

#ifndef RELEASE
    printf("Not release build\n");
#endif
```

### С выражениями

```c
#define VERSION 2

#if VERSION == 1
    printf("v1");
#elif VERSION == 2
    printf("v2");
#else
    printf("unknown");
#endif
```

### Комбинированные условия

```c
#if defined(DEBUG) && !defined(NDEBUG)
    // отладочный код
#endif

#if defined(_WIN32) || defined(__CYGWIN__)
    // Windows-код
#endif
```

## Применение

### 1. Платформенная зависимость

```c
#ifdef _WIN32
    #include <windows.h>
    #define PATH_SEPARATOR '\\'
#elif defined(__linux__)
    #include <unistd.h>
    #define PATH_SEPARATOR '/'
#elif defined(__APPLE__)
    #include <TargetConditionals.h>
    #define PATH_SEPARATOR '/'
#else
    #error "Unsupported platform"
#endif
```

### 2. Режимы сборки

```c
#ifdef NDEBUG
    // Release: без проверок
    #define ASSERT(x)
#else
    // Debug: с проверками
    #define ASSERT(x) \
        if (!(x)) { abort(); }
#endif
```

### 3. Версионирование API

```c
#if API_VERSION >= 2
    new_api_function();
#else
    legacy_api_function();
#endif
```

### 4. Оптимизации

```c
#ifdef ENABLE_SIMD
    #include <immintrin.h>
    #define COMPUTE simd_compute
#else
    #define COMPUTE scalar_compute
#endif
```

### 5. Экспорт символов

```c
#ifdef _WIN32
    #ifdef BUILDING_DLL
        #define EXPORT __declspec(dllexport)
    #else
        #define EXPORT __declspec(dllimport)
    #endif
#else
    #define EXPORT __attribute__((visibility("default")))
#endif

EXPORT void my_function();
```

## Предопределённые макросы

```c
// Платформа
#ifdef __linux__
#ifdef _WIN32
#ifdef __APPLE__

// Компилятор
#ifdef __GNUC__
#ifdef _MSC_VER
#ifdef __clang__

// Архитектура
#ifdef __x86_64__
#ifdef __aarch64__
#ifdef __i386__
```
