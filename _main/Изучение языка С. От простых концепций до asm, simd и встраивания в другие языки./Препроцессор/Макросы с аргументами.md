# Макросы с аргументами

**Функционоподобные макросы** — макросы, принимающие аргументы.

## Синтаксис

```c
#define MACRO(arg1, arg2) замена
```

## Примеры

### Простые макросы

```c
#define SQUARE(x) ((x) * (x))
#define MAX(a, b) ((a) > (b) ? (a) : (b))
#define MIN(a, b) ((a) < (b) ? (a) : (b))
#define ABS(x) ((x) < 0 ? -(x) : (x))

int s = SQUARE(5);      // 25
int m = MAX(3, 7);      // 7
int a = ABS(-5);        // 5
```

### Многострочные макросы

```c
#define FOR(i, n) \
    for (int i = 0; i < (n); i++)

#define SWAP(a, b) \
    do { \
        typeof(a) _tmp = (a); \
        (a) = (b); \
        (b) = _tmp; \
    } while (0)
```

### Макросы с variadic аргументами (C99)

```c
#define LOG(...) printf(__VA_ARGS__)
#define DEBUG(fmt, ...) \
    fprintf(stderr, "[DEBUG] " fmt, ##__VA_ARGS__)

LOG("Hello, %s!", "World");
DEBUG("Value: %d", x);
```

## Правила написания

### 1. Скобки вокруг аргументов

```c
// ПЛОХО
#define MUL(a, b) a * b
MUL(1 + 2, 3)  // 1 + 2 * 3 = 7, а не 9!

// ХОРОШО
#define MUL(a, b) ((a) * (b))
MUL(1 + 2, 3)  // ((1 + 2) * (3)) = 9
```

### 2. do-while(0) для многострочных

```c
// ПЛОХО
#define INIT() \
    x = 0; \
    y = 0;

if (cond)
    INIT();  // ошибка!
else { }

// ХОРОШО
#define INIT() \
    do { \
        x = 0; \
        y = 0; \
    } while (0)

if (cond)
    INIT();  // работает
else { }
```

### 3. Избегание побочных эффектов

```c
// ПРОБЛЕМА
#define SQUARE(x) ((x) * (x))
SQUARE(i++)  // ((i++) * (i++)) — i увеличивается дважды!

// РЕШЕНИЕ: использовать inline-функцию
inline int square(int x) { return x * x; }
```

## Макросы vs Функции

| Макросы | Функции |
|---------|---------|
| Быстрее (нет вызова) | Есть накладные расходы |
| Нет проверки типов | Проверка типов |
| Побочные эффекты | Безопасны |
| Отладка сложнее | Отладка проще |
