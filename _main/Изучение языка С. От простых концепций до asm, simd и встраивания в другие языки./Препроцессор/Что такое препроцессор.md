# Что такое препроцессор

**Препроцессор** — программа, обрабатывающая исходный код до начала компиляции.

## Что делает препроцессор

1. Удаляет комментарии
2. Обрабатывает директивы (`#define`, `#include` и др.)
3. Раскрывает макросы
4. Объединяет строковые литералы
5. Вставляет содержимое заголовочных файлов

## Директивы

Все директивы начинаются с `#`:

```c
#include <stdio.h>      // подключение заголовка
#define MAX 100         // определение макроса
#ifdef DEBUG            // условная компиляция
    printf("debug\n");
#endif
```

## Важные особенности

- Директивы **не заканчиваются** `;`
- Действуют до конца файла (если не ограничено)
- `#` должна быть в начале строки (после пробелов)

## Пример работы

```c
// Исходный код
#define PI 3.14
double r = PI * 2;

// После препроцессора
double r = 3.14 * 2;
```
