# Этапы компиляции C

Исходный код проходит несколько этапов перед получением исполняемого файла.

## Этапы

```
.c файл → Препроцессор → .i файл → Компилятор → .s файл → 
Ассемблер → .o файл → Компоновщик → исполняемый файл
```

## 1. Препроцессинг

```bash
gcc -E program.c -o program.i
```

- Обработка `#include`, `#define`, `#if`
- Удаление комментариев
- Раскрытие макросов

## 2. Компиляция

```bash
gcc -S program.i -o program.s
```

- Преобразование в ассемблерный код
- Оптимизации
- Проверка типов

## 3. Ассемблирование

```bash
gcc -c program.s -o program.o
```

- Преобразование в машинный код
- Получение объектного файла

## 4. Компоновка

```bash
gcc program.o -o program
```

- Связывание объектных файлов
- Разрешение внешних символов
- Добавление библиотек

## Просмотр этапов

```bash
# Только препроцессинг
gcc -E file.c

# Препроцессинг + компиляция
gcc -S file.c

# Всё кроме компоновки
gcc -c file.c
```
