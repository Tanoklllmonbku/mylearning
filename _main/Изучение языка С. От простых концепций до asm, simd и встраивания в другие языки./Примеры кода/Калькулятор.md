# Калькулятор

Консольный калькулятор из `_resources/base_level_C/calculator/` с поддержкой:
- Сложения, вычитания, умножения, деления
- Скобок
- Унарного минуса
- Вычисления выражений в инфиксной записи

## Структура проекта

```
calculator/
├── calculator.c      # основная логика парсинга и вычисления
├── calculator.h      # заголовок, макросы, структуры
├── sum.c / sum.h     # функция сложения
├── sub.c / sub.h     # функция вычитания
├── mul.c / mul.h     # функция умножения
└── div.c / div.h     # функция деления
```

### div.h / div.c

```c
// div.h
#ifndef DIV_H
#define DIV_H
#include <stdio.h>
int __DIV(int a, int b);
#endif

// div.c
#include "div.h"

int __DIV(int a, int b) {
    if (b == 0) {
        return 0;  // защита от деления на ноль
    }
    return a / b;
}
```

### calculator.c (основная логика)

Ключевые функции:

1. **delete_spaces** — удаление пробелов из строки
2. **parse_tokens** — парсинг строки в токены (числа и операторы)
3. **handle_unary_minus** — обработка унарного минуса
4. **find_deepest_paren** — поиск самой глубокой пары скобок
5. **evaluate_simple** — вычисление выражения без скобок
6. **get_result** — главная функция вычисления

## Компиляция

```bash
# Одной командой
gcc -Wall -O2 *.c -o calculator

# Раздельная компиляция
gcc -c sum.c -o sum.o
gcc -c sub.c -o sub.o
gcc -c mul.c -o mul.o
gcc -c div.c -o div.o
gcc -c calculator.c -o calculator.o
gcc *.o -o calculator
```

## Использование

```
Enter the expression:
2 + 2 * 2
Result: 6

Enter the expression:
(2 + 3) * 4
Result: 20

Enter the expression:
-5 + 3
Result: -2

Enter the expression:
10 / 2 + 3
Result: 8
```

## Алгоритм работы

1. Ввод выражения пользователем
2. Удаление пробелов
3. Парсинг в токены (числа и операторы)
4. Обработка унарных минусов
5. Вычисление скобок (с самых глубоких)
6. Вычисление финального выражения (* и / перед + и -)
7. Вывод результата

## Особенности

- Поддержка целых чисел
- Деление на ноль возвращает 0
- Унарный минус: `-5 + 3` = `-2`
- Скобки: `(2 + 3) * 4` = `20`
- Приоритет операций: `*` и `/` перед `+` и `-`
