# Auto vs Static

Демонстрация разницы между классами хранения `auto` и `static` из `_resources/base_level_C/auto_vs_static/`.

## Исходный код

```c
// auto_vs_static.c
#include <stdio.h>

void count() {
    // static - единожды регистрируется для всей программы
    static int x = 0;
    
    // auto - единожды для блока {}
    auto int y = 0;
    
    x++;
    y++;
    
    // Выведет: 1, 1   2, 1   3, 1
    printf("%d, %d\n", x, y);
}

int main() {
    // Счётчик
    int i = 0;
    for (i; i < 3; i++) {
        // Вызов функции для инициализации ++
        count();
    }
    
    // Возврат 0 при завершении работы программы - стандарт C
    // При ошибке - ненулевой код
    return 0;
}
```

## Компиляция

```bash
cd _resources/base_level_C/auto_vs_static
gcc auto_vs_static.c -o auto_vs_static
./auto_vs_static
```

## Вывод

```
1, 1
2, 1
3, 1
```

## Объяснение

| Переменная | Класс хранения | Где хранится | Инициализация | Время жизни |
|------------|---------------|--------------|---------------|-------------|
| `x` | `static` | Статическая память | Один раз при загрузке | Вся программа |
| `y` | `auto` | Стек | При каждом входе в блок | До выхода из блока |

### Почему такой вывод?

**Переменная `x` (static):**
- Инициализируется **один раз** при первом вызове
- Сохраняет значение между вызовами функции
- `x++` накапливает: 0→1, 1→2, 2→3

**Переменная `y` (auto):**
- Инициализируется **при каждом вызове** функции
- Уничтожается при выходе из блока
- `y++` всегда: 0→1

## Детали

### Static переменные

```c
static int x = 0;  // инициализация только один раз
```

- Хранятся в **статической области памяти**
- Сохраняют значение между вызовами
- Видны только внутри функции (локальная область)
- Инициализируются нулём по умолчанию, если не указано иное

### Auto переменные

```c
auto int y = 0;  // явное указание (избыточно)
int z = 0;       // неявно auto (рекомендуется)
```

- Хранятся в **стеке**
- Создаются при входе в блок `{}`
- Уничтожаются при выходе из блока
- **Не инициализируются** по умолчанию (неопределённое значение)
- Спецификатор `auto` избыточен в C (все локальные переменные — auto)

## Практическое применение

### Static для счётчиков

```c
void log_call() {
    static int call_count = 0;
    call_count++;
    printf("Вызов #%d\n", call_count);
}
```

### Static для кэширования

```c
int expensive_computation(int n) {
    static int cached_result = -1;
    static int cached_input = -1;
    
    if (n == cached_input && cached_result != -1) {
        return cached_result;  // возвращаем кэш
    }
    
    // дорогие вычисления
    cached_result = n * n;
    cached_input = n;
    return cached_result;
}
```
