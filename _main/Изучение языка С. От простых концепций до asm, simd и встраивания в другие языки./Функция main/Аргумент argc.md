# Аргумент `argc` (Argument Count)

## Определение

`argc` (argument count) — первый параметр функции `main`, представляющий **количество аргументов командной строки**, переданных программе при запуске.

```c
int main(int argc, char *argv[]) {
    // argc — количество аргументов
}
```

## Характеристики

### Тип данных

- **Тип**: `int`
- **Значение**: неотрицательное целое число (`argc >= 0`)
- **Семантика**: количество элементов в массиве `argv` минус один (последний элемент `NULL`)

### Значения

| Значение `argc` | Семантика |
|-----------------|-----------|
| `0` | Аргументы недоступны (редко, зависит от реализации) |
| `1` | Только имя программы (`argv[0]`) |
| `>1` | Имя программы + аргументы командной строки |

## Структура аргументов

Массив `argv` содержит `argc + 1` элементов, где:
- `argv[0]` … `argv[argc-1]` — указатели на строки-аргументы
- `argv[argc]` — гарантированно `NULL`

```
argc = 3

argv ──┬──> argv[0] ──> "./program" (Обязательно имя программы, под него резервируется превый векторный аргумент)
       ├──> argv[1] ──> "arg1"
       ├──> argv[2] ──> "arg2"
       └──> argv[3] ──> NULL  (гарантировано, указатель на конец массива из-за того, что массивы в C не могут знать своего точного размера)
```

## Примеры использования

### Минимальное значение

```c
#include <stdio.h>

int main(int argc, char *argv[]) {
    printf("argc = %d\n", argc);
    // При запуске без аргументов: argc = 1 (только имя программы)
    return 0;
}
```

**Вывод:**
```
$ ./program
argc = 1
```

### С аргументами

```c
#include <stdio.h>

int main(int argc, char *argv[]) {
    printf("Количество аргументов: %d\n", argc);
    for (int i = 0; i < argc; ++i) {
        printf("Аргумент %d: %s\n", i, argv[i]);
    }
    return 0;
}
```

**Вывод:**
```
$ ./program file1.txt file2.txt --verbose
Количество аргументов: 4
Аргумент 0: ./program
Аргумент 1: file1.txt
Аргумент 2: file2.txt
Аргумент 3: --verbose
```

## Важные свойства

### 1. Имя программы в `argv[0]`

`argv[0]` обычно содержит имя программы, но стандарт допускает:
- Пустую строку (`""`)
- Значение `NULL` (если `argc == 0`)
- Путь к исполняемому файлу

```c
if (argc > 0 && argv[0] != NULL) {
    printf("Имя программы: %s\n", argv[0]);
}
```

### 2. Гарантированный `NULL`-терминатор

Стандарт C гарантирует, что `argv[argc] == NULL`:

```c
// Безопасная итерация до NULL
for (int i = 0; i < argc; ++i) {
    // argv[i] валиден
}
// argv[argc] == NULL
```

### 3. Модифицируемость

Строки, на которые указывает `argv`, **модифицируемы**:

```c
// Можно изменять аргументы
if (argc > 1) {
    argv[1][0] = 'X';  // Изменение первого символа аргумента
}
```

Изменения сохраняются до завершения программы.

## Обработка аргументов

### Проверка количества аргументов

```c
#include <stdio.h>
#include <stdlib.h>

int main(int argc, char *argv[]) {
    if (argc < 2) {
        fprintf(stderr, "Ошибка: требуется хотя бы один аргумент\n");
        fprintf(stderr, "Использование: %s <файл>\n", argv[0]);
        return EXIT_FAILURE;
    }
    
    // Обработка аргументов
    printf("Обрабатываю файл: %s\n", argv[1]);
    return 0;
}
```

### Итерация по аргументам

```c
// Способ 1: по индексу
for (int i = 1; i < argc; ++i) {  // Начинаем с 1, пропускаем имя программы
    process_argument(argv[i]);
}

// Способ 2: через указатель
for (char **arg = argv + 1; arg < argv + argc; ++arg) {
    process_argument(*arg);
}
```

## Специфика платформ

### Unix/Linux

- Аргументы разделяются пробелами
- Специальные символы экранируются
- Оболочка обрабатывает подстановки (`*`, `?`)

```bash
$ ./program *.txt    # Оболочка раскроет * в список файлов
```

### Windows

- Аргументы передаются как строка
- Парсинг выполняется CRT
- Кавычки группируют аргументы с пробелами

```cmd
> program.exe "file name.txt"  # Один аргумент с пробелом
```

## Ошибки и ограничения

### Отрицательное значение

`argc` **не может** быть отрицательным:

```c
// argc всегда >= 0 по стандарту
if (argc < 0) {
    // Невозможно
}
```

### Выход за границы

```c
// ❌ ОПАСНО: выход за границы
for (int i = 0; i <= argc; ++i) {  // argv[argc] == NULL!
    printf("%s\n", argv[i]);       // Разыменование NULL при i == argc
}

// ✅ Безопасно
for (int i = 0; i < argc; ++i) {
    printf("%s\n", argv[i]);
}
```

## Связь с `argv`

`argc` и `argv` неразрывно связаны:

| Параметр | Роль |
|----------|------|
| `argc` | Количество аргументов (счётчик) |
| `argv` | Массив указателей на аргументы (вектор) |

```c
// Инвариант: argv[argc] == NULL
// Инвариант: argv[0..argc-1] — валидные указатели на строки
```
