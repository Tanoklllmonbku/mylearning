# Функция `exit` и альтернативы

## Функция `exit()`

### Объявление

```c
void exit(int status);  // <stdlib.h>
```

Функция `exit()` вызывает **нормальное завершение программы** с полной очисткой ресурсов.

### Поведение

При вызове `exit()` выполняются следующие действия **в порядке**:

1. **Обработчики `atexit`** — вызываются в обратном порядке регистрации
2. **Обработчики `at_quick_exit`** — не вызываются
3. **Сброс буферов потоков** — все буферизированные данные записываются
4. **Закрытие потоков** — все открытые потоки закрываются
5. **Удаление временных файлов** — созданных через `tmpfile()`
6. **Возврат управления** — среде выполнения с кодом `status`

### Код завершения

| Значение `status` | Макрос | Семантика |
|-------------------|--------|-----------|
| `0` | `EXIT_SUCCESS` | Успешное завершение |
| `EXIT_FAILURE` | `EXIT_FAILURE` | Неуспешное завершение |
| Другие | — | Реализуемо-зависимый статус |

### Пример

```c
#include <stdio.h>
#include <stdlib.h>

int main(void) {
    FILE *fp = fopen("data.txt", "r");
    
    if (fp == NULL) {
        fprintf(stderr, "Ошибка открытия файла\n");
        exit(EXIT_FAILURE);  // Принудительное завершение
    }
    
    // Обработка файла...
    fclose(fp);
    return EXIT_SUCCESS;
}
```

## Функция `_Exit()`

### Объявление

```c
// C99 - C11
void _Exit(int exit_code);

// C11 - C23
_Noreturn void _Exit(int exit_code);

// C23+
[[noreturn]] void _Exit(int exit_code);
```

### Поведение

`_Exit()` вызывает **немедленное завершение** без полной очистки:

| Действие | `_Exit()` |
|----------|-----------|
| Обработчики `atexit()` | ❌ Не вызывает |
| Обработчики `at_quick_exit()` | ❌ Не вызывает |
| Сброс буферов stdio | ⚠️ Зависит от реализации |
| Закрытие потоков | ⚠️ Зависит от реализации |
| Удаление временных файлов | ⚠️ Зависит от реализации |

### Когда использовать

- Критическая ошибка, когда очистка невозможна
- Требуется завершение без выполнения кода `atexit`
- Ситуации, где очистка может вызвать дополнительные проблемы

### Пример

```c
#include <stdlib.h>
#include <stdio.h>

void f1(void) { puts("Очистка 1"); }
void f2(void) { puts("Очистка 2"); }

int main(void) {
    printf("Запуск\n");
    atexit(f1);
    atexit(f2);
    fflush(stdout);  // _Exit может не сбросить буфер!
    _Exit(0);        // f1 и f2 НЕ вызовутся
}
```

**Вывод:**
```
Запуск
```

## Функция `quick_exit()` (C11)

### Объявление

```c
// C11 - C23
_Noreturn void quick_exit(int status);

// C23+
[[noreturn]] void quick_exit(int status);
```

### Поведение

`quick_exit()` — **быстрое завершение** с минимальной очисткой:

1. Вызывает функции, зарегистрированные через `at_quick_exit()` (в обратном порядке)
2. Вызывает `_Exit(status)` для завершения

| Действие | `quick_exit()` |
|----------|----------------|
| Обработчики `atexit()` | ❌ Не вызывает |
| Обработчики `at_quick_exit()` | ✅ Вызывает |
| Обработчики сигналов (`signal`) | ❌ Не вызывает |
| Полная очистка потоков | ❌ Не выполняет |

### Регистрация обработчиков

```c
int at_quick_exit(void (*func)(void));
```

Возвращает `0` при успехе, ненулевое значение при ошибке.

### Пример

```c
#include <stdio.h>
#include <stdlib.h>

void quick_cleanup(void) {
    printf("Быстрая очистка\n");
}

void normal_cleanup(void) {
    printf("Нормальная очистка\n");
}

int main(void) {
    atexit(normal_cleanup);      // НЕ вызовется
    at_quick_exit(quick_cleanup); // Вызовется
    
    printf("Запуск\n");
    quick_exit(0);
}
```

**Вывод:**
```
Запуск
Быстрая очистка
```

## Функция `abort()`

### Объявление

```c
// До C11
void abort(void);

// C11 - C23
_Noreturn void abort(void);

// C23+
[[noreturn]] void abort(void);
```

### Поведение

`abort()` вызывает **ненормальное завершение программы**:

1. Генерирует сигнал **SIGABRT**
2. Функции `atexit()` **не вызываются**
3. Очистка ресурсов — зависит от реализации
4. Возвращает статус **неуспешного завершения**

### Обработка сигнала

По умолчанию `abort()` завершает программу. Можно установить обработчик:

```c
#include <stdio.h>
#include <stdlib.h>
#include <signal.h>

void signal_handler(int sig) {
    printf("Получен сигнал %d\n", sig);
    // Если не вернуть управление — программа завершится
}

int main(void) {
    signal(SIGABRT, signal_handler);
    abort();  // Вызовет signal_handler
}
```

### Когда использовать

- Неустранимая ошибка внутреннего состояния
- Нарушение инвариантов программы (assertion failure)
- Ситуации, где продолжение работы небезопасно
- Отладка (вместо `exit()` для получения core dump)

### Пример

```c
#include <stdio.h>
#include <stdlib.h>

int main(void) {
    FILE *fp = fopen("critical.dat", "r");
    
    if (fp == NULL) {
        fprintf(stderr, "Критическая ошибка: файл не найден\n");
        abort();  // Аварийное завершение
    }
    
    fclose(fp);
    return 0;
}
```

## Сравнение функций завершения

| Характеристика | `exit()` | `_Exit()` | `quick_exit()` | `abort()` |
|----------------|----------|-----------|----------------|-----------|
| **Тип завершения** | Нормальное | Нормальное | Нормальное | Ненормальное |
| **`atexit()`** | ✅ Вызывает | ❌ Нет | ❌ Нет | ❌ Нет |
| **`at_quick_exit()`** | ❌ Нет | ❌ Нет | ✅ Вызывает | ❌ Нет |
| **Сброс буферов** | ✅ Да | ⚠️ Зависит | ❌ Нет | ⚠️ Зависит |
| **Закрытие потоков** | ✅ Да | ⚠️ Зависит | ❌ Нет | ⚠️ Зависит |
| **Сигнал** | Нет | Нет | Нет | SIGABRT |
| **Статус** | `status` | `status` | `status` | Неуспешный |
| **Стандарт** | C89+ | C99+ | C11+ | C89+ |

## Функция `atexit()`

### Объявление

```c
int atexit(void (*func)(void));  // <stdlib.h>
```

Регистрирует функцию `func`, которая будет вызвана при нормальном завершении программы.

### Характеристики

- **Порядок вызова**: обратный регистрации (LIFO — Last In, First Out)
- **Максимум обработчиков**: как минимум 32 (требование стандарта)
- **Возврат**: `0` при успехе, ненулевое значение при ошибке

### Пример

```c
#include <stdio.h>
#include <stdlib.h>

void f1(void) { printf("Очистка 1\n"); }
void f2(void) { printf("Очистка 2\n"); }
void f3(void) { printf("Очистка 3\n"); }

int main(void) {
    atexit(f1);  // Зарегистрирована первой
    atexit(f2);  // Зарегистрирована второй
    atexit(f3);  // Зарегистрирована третьей
    
    printf("Основная программа\n");
    
    // При завершении вызовутся в порядке: f3, f2, f1
    return 0;
}
```

**Вывод:**
```
Основная программа
Очистка 3
Очистка 2
Очистка 1
```

## Сводная таблица

| Сценарий | Рекомендуемая функция |
|----------|----------------------|
| Успешное завершение | `return 0` или `exit(EXIT_SUCCESS)` |
| Ошибка с очисткой | `exit(EXIT_FAILURE)` |
| Критическая ошибка (без очистки) | `_Exit(EXIT_FAILURE)` |
| Быстрое завершение с мин. очисткой | `quick_exit(status)` (C11+) |
| Аварийное завершение (баг) | `abort()` |
