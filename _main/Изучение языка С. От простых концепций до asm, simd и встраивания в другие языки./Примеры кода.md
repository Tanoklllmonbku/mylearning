### Структура проектов
```
_resources/
├── base_level_C/
│   ├── hello_world/
│   │   └── hello_world.c
│   └── calculator/
│       ├── calculator.c      # основная логика
│       ├── calculator.h      # заголовок
│       ├── sum.c / sum.h     # сложение
│       ├── sub.c / sub.h     # вычитание
│       ├── mul.c / mul.h     # умножение
│       └── div.c / div.h     # деление
```

## Компиляция примеров

### Hello World

```bash
cd _resources/base_level_C/hello_world
gcc hello_world.c -o hello_world
./hello_world
```

**Используемые концепции:**
- [[stdio.h]] — `printf()`, `#include`
- [[Функция main в C]] — точка входа
- [[_main/Изучение языка С. От простых концепций до asm, simd и встраивания в другие языки./Ключевые слова/int]] — тип возвращаемого значения
- [[return]] — возврат значения
- [[Компиляция одного файла]] — сборка

### Калькулятор

```bash
cd _resources/base_level_C/calculator

# Компиляция всех файлов
gcc -Wall -O2 *.c -o calculator

# Или раздельная компиляция
gcc -c sum.c -o sum.o
gcc -c sub.c -o sub.o
gcc -c mul.c -o mul.o
gcc -c div.c -o div.o
gcc -c calculator.c -o calculator.o
gcc *.o -o calculator

# Запуск
./calculator
```

**Используемые концепции:**
- [[define]] — макросы `MAX_INPUT_LEN`, `MAX_TOKENS`
- [[_main/Изучение языка С. От простых концепций до asm, simd и встраивания в другие языки./Заголовочные файлы/Include guards]] — защита от повторного включения
- [[typedef]] и [[struct]] — структура `_CHAR_OPS`
- [[for]] — циклы парсинга
- [[sizeof]] — вычисление размеров
- [[Компиляция нескольких файлов]] — многофайловая сборка
- [[Примеры сборки проектов]] — примеры Makefile

### Auto vs Static

```bash
cd _resources/base_level_C/auto_vs_static
gcc auto_vs_static.c -o auto_vs_static
./auto_vs_static
```

**Вывод:**
```
1, 1
2, 1
3, 1
```

**Используемые концепции:**
- [[auto]] — автоматические переменные в стеке
- [[static]] — сохранение значения между вызовами
- [[for]] — цикл для демонстрации
- [[stdio.h]] — `printf()`
- [[Компиляция одного файла]] — сборка

### Thread Local

```bash
cd _resources/base_level_C/thread_local
gcc -std=c11 thread_local.c -o thread_local -lpthread
./thread_local
```

**Вывод:**
```
Start counting from: 0
Start counting from: 0
H
E
L
...
```

**Используемые концепции:**
- [[_Thread_local]] — уникальная переменная на поток
- [[threads.h]] — `thrd_create()`, `thrd_join()`
- [[struct]] и [[typedef]] — передача данных в поток
- [[Компиляция одного файла]] — сборка с `-lpthread`

**Важно:** `thread_local` защищает только саму переменную, но не данные по указателю!
