# Токены C

Токены — минимальные лексические единицы исходного кода на C. Компилятор разбирает код на токены на этапе лексического анализа.

## Типы токенов

### 1. Идентификаторы
Имена переменных, функций, типов, меток.
- Начинаются с буквы или `_`
- Содержат буквы, цифры, `_`
- Регистрозависимые

```c
int counter;
void my_function();
#define _DEBUG 1
```

### 2. Ключевые слова
Зарезервированные слова языка (C99/C11):

| Категория | Ключевые слова |
|-----------|----------------|
| Типы | `int`, `char`, `float`, `double`, `void`, `_Bool`, `_Complex`, `_Imaginary` |
| Модификаторы | `signed`, `unsigned`, `short`, `long`, `const`, `volatile`, `restrict`, `_Atomic` |
| Хранения | `auto`, `register`, `static`, `extern`, `typedef`, `thread_local` |
| Управление | `if`, `else`, `switch`, `case`, `default`, `for`, `while`, `do`, `goto`, `continue`, `break`, `return` |
| Прочие | `struct`, `union`, `enum`, `sizeof`, `alignof`, `inline`, `noreturn`, `_Static_assert` |

### 3. Литералы (константы)

```c
// Целочисленные
42
0x2A        // шестнадцатеричный
052         // восьмеричный
0b101010    // двоичный (C23)
100U        // unsigned
100L        // long

// Вещественные
3.14
2.5e-3      // научная нотация
1.0f        // float
1.0L        // long double

// Символьные
'A'
'\n'        // escape-последовательность
L'宽'       // широкий символ

// Строковые
"hello"
u"unicode"  // UTF-16
U"unicode"  // UTF-32
u8"UTF-8"
```

### 4. Операторы

```c
// Арифметические
+  -  *  /  %  ++  --  unary +  unary -

// Отношения
==  !=  <  >  <=  >=

// Логические
&&  ||  !

// Побитовые
&  |  ^  ~  <<  >>

// Присваивание
=  +=  -=  *=  /=  %=  &=  |=  ^=  <<=  >>=

// Другие
?:  ,  .  ->  []  ()  sizeof  _Alignof  _Generic
```

### 5. Разделители

```c
;       // окончание утверждения
,       // разделитель списков
:       // метка, case
=       // инициализация
()      // вызов функции, группировка
[]      // индексация
{}      // блок кода, инициализатор
#       // директива препроцессора
...     // вариадические аргументы
```

## Важные замечания

- **Пробелы и табуляции** игнорируются (кроме разделения токенов)
- **Комментарии** (`/* */` и `//`) удаляются на этапе лексического анализа
- **Макросы препроцессора** обрабатываются до лексического анализа
- **Дигraphs и триграфы** заменяются до токенизации (устаревшие конструкции)

## Пример токенизации

```c
int x = 42 + y;
```

Разбивается на токены:
1. `int` (ключевое слово)
2. `x` (идентификатор)
3. `=` (оператор)
4. `42` (литерал)
5. `+` (оператор)
6. `y` (идентификатор)
7. `;` (разделитель)
