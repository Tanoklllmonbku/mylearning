# Типичные ошибки компиляции

Распространённые ошибки и способы их решения.

## Ошибки линковки

### Undefined reference

```
undefined reference to `function_name'
```

**Причины:**
- Функция не определена
- Забыли подключить библиотеку
- Неправильный порядок файлов

**Решение:**
```bash
# Проверить наличие функции
nm *.o | grep function_name

# Подключить библиотеку
gcc main.c -lm -o program

# Правильный порядок
gcc main.c utils.c -o program
```

### Multiple definition

```
multiple definition of `function_name'
```

**Причины:**
- Функция определена в заголовке
- Дублирование файлов

**Решение:**
```c
// В заголовке — объявление
void function(void);

// В .c — определение
void function(void) { }
```

## Ошибки препроцессора

### File not found

```
fatal error: header.h: No such file or directory
```

**Решение:**
```bash
# Указать путь
gcc -I./include main.c -o program

# Проверить путь
ls include/header.h
```

### Macro redefined

```
warning: 'MACRO' redefined
```

**Решение:**
```c
// Использовать #undef
#undef MACRO
#define MACRO value

// Или проверить
#ifndef MACRO
#define MACRO value
#endif
```

## Ошибки типов

### Implicit declaration

```
warning: implicit declaration of function 'func'
```

**Причины:**
- Нет #include
- Нет объявления функции

**Решение:**
```c
#include "header.h"  // добавить заголовок

// Или объявить
int func(int x);
```

### Incompatible pointer type

```
warning: passing argument from incompatible pointer type
```

**Решение:**
```c
// Неправильно
void func(int* p);
int* x;
func((char*)x);  // warning

// Правильно
func(x);
```

## Ошибки стандарта

### Unknown type

```
error: unknown type name 'bool'
```

**Решение:**
```bash
# Указать стандарт
gcc -std=c99 main.c -o program

# Или включить расширения
gcc -std=gnu11 main.c -o program
```

### Variable length array

```
warning: variable length arrays are a C99 feature
```

**Решение:**
```bash
gcc -std=c99 main.c -o program
```

## Предупреждения

### Unused variable

```
warning: unused variable 'x'
```

**Решение:**
```c
// Использовать переменную
int x = 5;
printf("%d\n", x);

// Или удалить
// int x = 5;

// Или подавить (void)
int x = 5;
(void)x;
```

### Format specifies

```
warning: format specifies type 'int' but argument has type 'long'
```

**Решение:**
```c
// Неправильно
long x = 100;
printf("%d\n", x);

// Правильно
printf("%ld\n", x);
```

### Control reaches end of non-void function

```
warning: control reaches end of non-void function
```

**Решение:**
```c
// Добавить return
int func() {
    return 0;
}
```

## Ошибки сборки

### Permission denied

```
/usr/bin/ld: cannot open output file program: Permission denied
```

**Решение:**
```bash
# Проверить права
ls -la

# Запуск от нужного пользователя
# Или изменить путь вывода
gcc main.c -o /tmp/program
```

### Library not found

```
/usr/bin/ld: cannot find -lmylib
```

**Решение:**
```bash
# Проверить наличие
ls /usr/lib/libmylib.*

# Указать путь
gcc -L/path/to/lib main.c -lmylib -o program

# Установить библиотеку
sudo apt install libmylib-dev
```

## Отладка ошибок

```bash
# Подробный вывод
gcc -v main.c -o program

# Показать предупреждения с опциями
gcc -fdiagnostics-show-option main.c

# Статический анализ
gcc -fanalyzer main.c
```
