# Компиляция нескольких файлов

Компиляция программ, состоящих из нескольких исходных файлов.

## Структура проекта

```
project/
├── main.c
├── utils.c
├── utils.h
└── math_ops.c
```

## Пример файлов

```c
// utils.h
#ifndef UTILS_H
#define UTILS_H

void print_message(const char* msg);
int add(int a, int b);

#endif
```

```c
// utils.c
#include "utils.h"
#include <stdio.h>

void print_message(const char* msg) {
    printf("%s\n", msg);
}

int add(int a, int b) {
    return a + b;
}
```

```c
// main.c
#include "utils.h"

int main() {
    print_message("Hello");
    int result = add(2, 3);
    return 0;
}
```

## Способ 1: Одной командой

```bash
gcc main.c utils.c -o program
```

**Преимущества:**
- Простота
- Автоматическая пересборка всех файлов

**Недостатки:**
- Компилирует всё каждый раз
- Медленно для больших проектов

## Способ 2: Раздельная компиляция

```bash
# Компиляция каждого файла в объектный
gcc -c main.c -o main.o
gcc -c utils.c -o utils.o

# Линковка объектных файлов
gcc main.o utils.o -o program
```

**Преимущества:**
- Компилируются только изменённые файлы
- Быстрее для больших проектов

**Недостатки:**
- Больше команд

## Способ 3: Автоматическая зависимость

```bash
# Генерация зависимостей
gcc -MM main.c
# Вывод: main.o: main.c utils.h

# Компиляция с зависимостями
gcc -MMD -c main.c -o main.o
# Создаёт main.d с зависимостями
```

## Пересборка при изменении

```bash
# Изменили utils.c
gcc -c utils.c -o utils.o  # перекомпилировали только utils
gcc main.o utils.o -o program  # перелинковали
```

## Работа с заголовками

```bash
# Указание директорий с заголовками
gcc -I./include main.c utils.c -o program

# Несколько директорий
gcc -I./include -I../lib/include main.c -o program
```

## Пример с тремя файлами

```bash
# Файлы: main.c, utils.c, math_ops.c

# Одной командой
gcc main.c utils.c math_ops.c -o program

# Раздельно
gcc -c main.c -o main.o
gcc -c utils.c -o utils.o
gcc -c math_ops.c -o math_ops.o
gcc main.o utils.o math_ops.o -o program
```

## Пример из практики

Готовый проект в [[Калькулятор]]:

```
calculator/
├── calculator.c      # основная логика
├── calculator.h      # заголовок
├── sum.c / sum.h     # сложение
├── sub.c / sub.h     # вычитание
├── mul.c / mul.h     # умножение
└── div.c / div.h     # деление
```

**Компиляция:**

```bash
cd _resources/base_level_C/calculator

# Одной командой
gcc -Wall -O2 *.c -o calculator

# Раздельная компиляция
gcc -c sum.c -o sum.o
gcc -c sub.c -o sub.o
gcc -c mul.c -o mul.o
gcc -c div.c -o div.o
gcc -c calculator.c -o calculator.o
gcc *.o -o calculator
```

**Запуск:**
```
./calculator
```

```bash
# Символы в исполняемом файле

```
