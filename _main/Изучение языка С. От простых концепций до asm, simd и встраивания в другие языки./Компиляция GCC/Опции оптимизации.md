# Опции оптимизации

Уровни и типы оптимизации GCC.

## Уровни оптимизации

### -O0: Без оптимизации

```bash
gcc -O0 program.c -o program
```

- По умолчанию
- Быстрая компиляция
- Легче отлаживать
- **Для отладки**

### -O1: Базовая оптимизация

```bash
gcc -O1 program.c -o program
```

- Уменьшает размер и время
- Быстрая компиляция
- **Для разработки**

### -O2: Рекомендуемая

```bash
gcc -O2 program.c -o program
```

- Больше оптимизаций
- Увеличивает время компиляции
- **Для релиза**

### -O3: Агрессивная

```bash
gcc -O3 program.c -o program
```

- Векторизация циклов
- Больше инлайнинга
- Может увеличить размер
- **Для вычислений**

### -Os: Оптимизация по размеру

```bash
gcc -Os program.c -o program
```

- Минимальный размер
- Может быть медленнее
- **Для встраиваемых систем**

### -Ofast: Быстрая (не стандарт)

```bash
gcc -Ofast program.c -o program
```

- Как -O3 + быстрые математические функции
- Может нарушать IEEE 754
- **Только если допустимы неточности**

## Специфичные оптимизации

### Инлайнинг функций

```bash
gcc -finline-functions program.c -o program
```

### Векторизация

```bash
gcc -ftree-vectorize program.c -o program
gcc -msse4.2 program.c -o program  # SSE
gcc -mavx2 program.c -o program    # AVX
```

### Размотка циклов

```bash
gcc -funroll-loops program.c -o program
```

### Оптимизация по профилю

```bash
# Сбор профиля
gcc -fprofile-arcs -ftest-coverage program.c -o program
./program  # запуск для сбора данных

# Использование профиля
gcc -fprofile-use program.c -o program
```

## Архитектурные оптимизации

### Для текущей машины

```bash
gcc -march=native program.c -o program
```

### Конкретная архитектура

```bash
gcc -march=x86-64-v3 program.c -o program
gcc -mtune=skylake program.c -o program
```

## Сравнение размеров

```bash
gcc -O0 program.c -o program-O0
gcc -O2 program.c -o program-O2
gcc -Os program.c -o program-Os

ls -lh program-*
```

## Сравнение скорости

```bash
time ./program-O0
time ./program-O2
time ./program-O3
```

## Рекомендации

| Цель | Опции |
|------|-------|
| Отладка | `-O0 -g` |
| Разработка | `-O1 -Wall` |
| Релиз | `-O2 -Wall` |
| Макс. скорость | `-O3 -march=native` |
| Мин. размер | `-Os` |
