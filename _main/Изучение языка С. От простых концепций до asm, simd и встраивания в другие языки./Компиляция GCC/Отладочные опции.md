# Отладочные опции

Опции GCC для отладки программ.

## -g: Отладочная информация

```bash
gcc -g program.c -o program
```

Добавляет отладочную информацию:
- Номера строк
- Имена переменных
- Типы данных

**Использование с gdb:**
```bash
gdb ./program
(gdb) break main
(gdb) run
(gdb) print variable
```

## Уровни отладочной информации

### -g0

```bash
gcc -g0 program.c -o program
```

Без отладочной информации (как без -g).

### -g1

```bash
gcc -g1 program.c -o program
```

Минимальная информация (только для gdb).

### -g2

```bash
gcc -g2 program.c -o program
```

По умолчанию. Полная информация.

### -g3

```bash
gcc -g3 program.c -o program
```

Дополнительная информация о макросах.

## Комбинация с оптимизацией

```bash
# Отладка без оптимизации
gcc -g -O0 program.c -o program

# Отладка с оптимизацией (сложнее)
gcc -g -O2 program.c -o program
```

## -ggdb: Для GDB

```bash
gcc -ggdb program.c -o program
```

Специфичная информация для GDB.

## -save-temps: Сохранение промежуточных файлов

```bash
gcc -save-temps program.c -o program
```

Создаёт файлы:
- `program.i` — после препроцессора
- `program.s` — ассемблерный код
- `program.o` — объектный файл

## -E: Только препроцессинг

```bash
gcc -E program.c -o program.i
```

Проверка макросов и #include.

## -S: Только компиляция

```bash
gcc -S program.c -o program.s
```

Просмотр ассемблерного кода.

## -c: Только компиляция в объект

```bash
gcc -c program.c -o program.o
```

Без линковки.

## -v: Подробный вывод

```bash
gcc -v program.c -o program
```

Показывает этапы компиляции и пути.

## -H: Трассировка заголовков

```bash
gcc -H program.c -o program
```

Показывает цепочку включения заголовков.

```
. stdio.h
.. bits/libc-header-start.h
.. bits/types.h
...
```

## -fdiagnostics: Диагностика

```bash
# Показать опции предупреждений
gcc -fdiagnostics-show-option program.c

# Цветной вывод
gcc -fdiagnostics-color=always program.c

# Дополнительная информация
gcc -fdiagnostics-details program.c
```

## Пример отладки

```c
// debug.c
#include <stdio.h>

int factorial(int n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}

int main() {
    int x = 5;
    int result = factorial(x);
    printf("%d! = %d\n", x, result);
    return 0;
}
```

```bash
# Компиляция для отладки
gcc -g -O0 -Wall debug.c -o debug

# Запуск в gdb
gdb ./debug
(gdb) break factorial
(gdb) run
(gdb) next
(gdb) print n
(gdb) backtrace
```

## Рекомендации

| Цель | Опции |
|------|-------|
| Отладка | `-g -O0 -Wall` |
| Профилирование | `-g -O2` |
| Релиз | `-O2` (без -g) |
