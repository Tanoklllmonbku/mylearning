# Опции предупреждений

Настройка предупреждений компилятора GCC.

## Базовые предупреждения

### -Wall

```bash
gcc -Wall program.c -o program
```

Включает основные предупреждения:
- Неиспользуемые переменные
- Неинициализированные переменные
- Несоответствие форматов
- Пропущенные return

### -Wextra

```bash
gcc -Wextra program.c -o program
```

Дополнительные предупреждения:
- Неиспользуемые параметры
- Знак/беззнак сравнения
- Ложные условия

### -Wpedantic

```bash
gcc -Wpedantic program.c -o program
```

Предупреждения о нарушениях стандарта C.

## Рекомендуемая комбинация

```bash
gcc -Wall -Wextra -Wpedantic program.c -o program
```

## Специфичные предупреждения

### Неиспользуемое

```bash
-Wunused           # всё неиспользуемое
-Wunused-variable  # переменные
-Wunused-function  # функции
-Wunused-parameter # параметры
-Wunused-value     # значения
```

### Инициализация

```bash
-Wuninitialized    # неинициализированные
-Wmaybe-uninitialized  # возможно неинициализированные
```

### Формат

```bash
-Wformat           # формат printf/scanf
-Wformat=2         # строгий формат
-Wformat-security  # безопасность формата
```

### Указатели

```bash
-Wnull-dereference  # разыменование NULL
-Wpointer-arith     # арифметика указателей
```

### Типы

```bash
-Wsign-compare     # знаковые/беззнаковые
-Wconversion       # неявные преобразования
-Wcast-align       # выравнивание при приведении
```

## -Werror: Ошибки вместо предупреждений

```bash
gcc -Wall -Werror program.c -o program
```

Все предупреждения становятся ошибками.

**Для CI/CD:**
```bash
gcc -Wall -Wextra -Werror -pedantic program.c -o program
```

## Отключение предупреждений

```bash
# Отключить конкретное
gcc -Wall -Wno-unused-parameter program.c -o program

# Отключить все
gcc -w program.c -o program
```

## Прагмы для отключения

```c
// В коде
#pragma GCC diagnostic push
#pragma GCC diagnostic ignored "-Wunused-variable"
// код с предупреждениями
#pragma GCC diagnostic pop
```

## Примеры предупреждений

### Неиспользуемая переменная

```c
int main() {
    int x = 5;  // warning: unused variable
    return 0;
}
```

### Несоответствие формата

```c
int x = 42;
printf("%s\n", x);  // warning: format '%s' expects 'char *'
```

### Пропущенный return

```c
int func() {
    // warning: control reaches end of non-void function
}
```

## Статический анализ

```bash
# Встроенный анализ
gcc -fanalyzer program.c -o program

# С инлайн-советами
gcc -fdiagnostics-show-option program.c
```

## Рекомендации

| Этап | Опции |
|------|-------|
| Разработка | `-Wall -Wextra` |
| Production | `-Wall -Wextra -Werror` |
| Строгий | `-Wall -Wextra -Wpedantic -Werror` |
