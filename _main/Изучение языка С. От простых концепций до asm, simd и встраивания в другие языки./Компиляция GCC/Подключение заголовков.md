# Подключение заголовков

Подключение заголовочных файлов при компиляции.

## Формы #include

```c
#include <stdio.h>    // системные заголовки
#include "mylib.h"    // пользовательские заголовки
```

## Различия

| Форма | Где ищет |
|-------|----------|
| `<file.h>` | Системные директории компилятора |
| `"file.h"` | Текущая директория → системные |

## Указание путей к заголовкам

### Флаг `-I`

```bash
# Заголовки в текущей папке
gcc -I. main.c -o program

# Заголовки в папке include
gcc -I./include main.c -o program

# Несколько путей
gcc -I./include -I../lib/include main.c -o program

# Абсолютный путь
gcc -I/usr/local/include main.c -o program
```

### Флаг `-isystem`

```bash
# Системные пути (предупреждения отключены)
gcc -isystem./custom/include main.c -o program
```

## Пример структуры проекта

```
project/
├── src/
│   ├── main.c
│   └── utils.c
├── include/
│   ├── utils.h
│   └── config.h
└── build/
```

### Компиляция

```bash
# Указание пути к заголовкам
gcc -I./include src/main.c src/utils.c -o build/program

# С раздельной компиляцией
gcc -I./include -c src/main.c -o build/main.o
gcc -I./include -c src/utils.c -o build/utils.o
gcc build/main.o build/utils.o -o build/program
```

## Поиск библиотек

### Флаг `-L`

```bash
# Поиск библиотек в директории
gcc -L./lib main.c -lmylib -o program
```

### Флаг `-l`

```bash
# Подключение библиотеки
gcc main.c -lm -o program        # math library
gcc main.c -lpthread -o program  # pthread library
```

## Переменные окружения

```bash
# Включение путей через окружение
export CPATH=./include:/usr/local/include
export LIBRARY_PATH=./lib:/usr/local/lib
export LD_LIBRARY_PATH=./lib

gcc main.c -o program
```

## Порядок флагов важен

```bash
# ПРАВИЛЬНО: библиотеки после исходных файлов
gcc main.c -lm -o program

# НЕПРАВИЛЬНО: библиотека до файла
gcc -lm main.c -o program  # может не сработать
```

## Пример с математикой

```c
// main.c
#include <math.h>
#include "mylib.h"

int main() {
    double x = sqrt(2.0);
    return 0;
}
```

```bash
# Компиляция
gcc -I. main.c -o program -lm
```
