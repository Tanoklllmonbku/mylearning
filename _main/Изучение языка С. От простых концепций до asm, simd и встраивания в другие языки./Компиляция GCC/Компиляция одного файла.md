# Компиляция одного файла

Базовая компляция программы из одного исходного файла.

## Базовая компиляция

```bash
gcc program.c -o program
```

**Что происходит:**
1. Препроцессинг
2. Компиляция
3. Ассемблирование
4. Линковка

**Результат:** исполняемый файл `program`

## Запуск программы

```bash
./program  # Linux/macOS
program.exe  # Windows
```

## Опции компиляции

### Имя выходного файла

```bash
gcc program.c -o myapp
./myapp
```

### Стандарт C

```bash
gcc -std=c99 program.c -o program
gcc -std=c11 program.c -o program
gcc -std=c17 program.c -o program
gcc -std=c23 program.c -o program  # экспериментальный
```

### Предупреждения

```bash
gcc -Wall program.c -o program      # основные предупреждения
gcc -Wextra program.c -o program    # дополнительные
gcc -Werror program.c -o program    # ошибки вместо предупреждений
gcc -Wall -Wextra -Werror program.c -o program  # строго
```

### Оптимизация

```bash
gcc -O0 program.c -o program   # без оптимизации (отладка)
gcc -O1 program.c -o program   # базовая оптимизация
gcc -O2 program.c -o program   # рекомендуемая
gcc -O3 program.c -o program   # агрессивная
gcc -Os program.c -o program   # оптимизация по размеру
```

### Отладка

```bash
gcc -g program.c -o program    # отладочная информация
gdb ./program                  # запуск отладчика
```

## Примеры

### Простая программа

```c
// hello.c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

```bash
gcc hello.c -o hello
./hello
```

### Пример из практики

Готовый пример в [[Hello World]]:

```bash
cd _resources/base_level_C/hello_world
gcc hello_world.c -o hello_world
./hello_world
```

**Вывод:**
```
Hello, World!
```

### С математикой

```c
// math.c
#include <stdio.h>
#include <math.h>

int main() {
    double x = sqrt(2.0);
    printf("%f\n", x);
    return 0;
}
```

```bash
gcc math.c -o math -lm  # -lm подключает math library
./math
```

### Со всеми предупреждениями

```bash
gcc -Wall -Wextra -pedantic program.c -o program
```

### Для отладки

```bash
gcc -g -O0 -Wall program.c -o program
gdb ./program
```

### Для продакшена

```bash
gcc -O2 -Wall program.c -o program
```
