# Что такое выравнивание

**Выравнивание** — требование размещения объектов в памяти по адресам, кратным определённому значению (степени двойки).

## Зачем нужно выравнивание

1. **Требования процессора** — некоторые архитектуры не могут читать данные по невыровненным адресам
2. **Производительность** — выровненный доступ быстрее (один цикл памяти вместо двух)
3. **Атомарность** — атомарные операции требуют выравнивания

## Пример

```c
// На x86-64:
char c;      // 1 байт, выравнивание 1
short s;     // 2 байта, выравнивание 2
int i;       // 4 байта, выравнивание 4
double d;    // 8 байт, выравнивание 8

// Адреса в памяти:
// c: 0x1000 (кратен 1) ✓
// s: 0x1002 (кратен 2) ✓
// i: 0x1008 (кратен 4) ✓
// d: 0x1010 (кратен 8) ✓
```

## Проверка выравнивания

```c
#include <stdalign.h>
#include <stdio.h>

printf("%zu\n", alignof(int));     // 4
printf("%zu\n", _Alignof(double)); // 8
```
