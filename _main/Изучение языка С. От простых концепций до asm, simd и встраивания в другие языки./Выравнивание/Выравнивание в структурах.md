# Выравнивание в структурах

Структуры выравниваются по **наибольшему требованию** своего члена.

## Правила

1. Каждый член размещается со своим выравниванием
2. Между членами добавляются **отступы** (padding)
3. Структура выравнивается по максимальному члену

## Пример

```c
struct Bad {
    char c;    // 1 байт + 3 байта padding
    int i;     // 4 байта
    short s;   // 2 байта + 2 байта padding
};             // Итого: 12 байт (не 7!)

struct Good {
    int i;     // 4 байта
    short s;   // 2 байта
    char c;    // 1 байт + 1 байт padding
};             // Итого: 8 байт
```

## Проверка размера и выравнивания

```c
printf("%zu\n", sizeof(struct Bad));   // 12
printf("%zu\n", alignof(struct Bad));  // 4 (по int)

printf("%zu\n", sizeof(struct Good));  // 8
printf("%zu\n", alignof(struct Good)); // 4
```

## Упаковка структур (непереносимо)

```c
// GCC/Clang
struct __attribute__((packed)) Packed {
    char c;
    int i;
    short s;
};  // 7 байт, но медленный доступ!

// MSVC
#pragma pack(push, 1)
struct Packed { ... };
#pragma pack(pop)
```
