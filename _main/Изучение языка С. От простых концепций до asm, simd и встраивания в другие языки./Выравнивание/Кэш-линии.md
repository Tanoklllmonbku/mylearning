# Кэш-линии

**Кэш-линия** — минимальная единица данных, передаваемая между кэшем и памятью.

## Характеристики

| Уровень кэша | Размер кэш-линии |
|--------------|------------------|
| L1 | 64 байта (типично) |
| L2 | 64 байта |
| L3 | 64 байта |

## Выравнивание по кэш-линии

```c
// Структура, занимающая одну кэш-линию
struct _Alignas(64) CacheAligned {
    int counter;
    char padding[60];  // до 64 байт
};

// Разделение данных для разных потоков
struct _Alignas(64) ThreadData {
    int local_counter;
    char pad[60];  // предотвращает false sharing
};

Thread_data thread1, thread2;  // в разных кэш-линиях
```

## False Sharing (Ложное разделение)

```c
// ПРОБЛЕМА: два потока пишут в одну кэш-линию
struct Bad {
    int thread1_data;  // поток 1 пишет
    int thread2_data;  // поток 2 пишет
};  // 8 байт в одной кэш-линии!

// РЕШЕНИЕ: разделение по кэш-линиям
struct Good {
    int thread1_data;
    char pad1[60];
    int thread2_data;
    char pad2[60];
};  // 128 байт, разные кэш-линии
```

## Применение

- Многопоточные счётчики
- Lock-free структуры
- Высокопроизводительные буферы
