# Выравнивание типов

Каждый тип в C имеет **естественное выравнивание** — требование к адресу размещения.

## Выравнивание фундаментальных типов (x86-64)

| Тип | Размер | Выравнивание |
|-----|--------|--------------|
| `char` | 1 | 1 |
| `short` | 2 | 2 |
| `int` | 4 | 4 |
| `long` | 8 | 8 |
| `float` | 4 | 4 |
| `double` | 8 | 8 |
| `long double` | 16 | 16 |
| Указатель | 8 | 8 |

## Проверка

```c
#include <stdalign.h>

printf("%zu\n", alignof(char));        // 1
printf("%zu\n", alignof(int));         // 4
printf("%zu\n", alignof(void*));       // 8
printf("%zu\n", alignof(max_align_t)); // 16 или 32
```

## Выравнивание массивов

```c
int arr[10];  // выравнивание как у int (4 байта)
```
