# _Alignas

**Категория:** Спецификатор выравнивания (C11)

## Назначение

Запрос пользовательского выравнивания для объекта или типа.

## Синтаксис

```c
_Alignas(выравнивание) тип имя;
_Alignas(тип) тип имя;
```

## Пример

```c
#include <stdalign.h>

// Выравнивание константой
_Alignas(16) int x;
alignas(32) double array[10];  // макрос из stdalign.h

// Выравнивание типом
_Alignas(max_align_t) char buffer[256];

// В структуре
struct _Alignas(64) CacheLine {
    int data[16];
};

// Несколько спецификаторов (максимальное)
_Alignas(16) _Alignas(32) int y;  // выравнивание 32
```

## Ограничения

- Выравнивание должно быть **степенью двойки**
- Нельзя указать выравнивание **меньше естественного**
- Максимальное выравнивание: `alignof(max_align_t)`

## Применение

- SIMD типы данных (SSE, AVX)
- Атомарные операции
- DMA буферы
- Кэш-линии
