# Атомарные операции и выравнивание

Атомарные типы требуют выравнивания для корректной работы.

## Требования

- `_Atomic(T)` имеет то же выравнивание, что и `T`
- Атомарные операции требуют **естественного выравнивания**

## Пример

```c
#include <stdatomic.h>
#include <stdio.h>

_Atomic int atomic_counter;  // выравнивание 4
_Atomic long long atomic_64; // выравнивание 8

printf("%zu\n", alignof(_Atomic(int)));        // 4
printf("%zu\n", alignof(_Atomic(long long)));  // 8
```

## Проблемы при неправильном выравнивании

```c
// Структура с потенциальной проблемой
struct Bad {
    char flag;              // 1 байт
    _Atomic int counter;    // требует 4 байта!
};                          // padding между полями

// Правильно
struct Good {
    _Atomic int counter;    // сначала атомики
    char flag;              // потом мелкие типы
};
```

## Lock-free атомики

```c
#include <stdatomic.h>

// Проверка lock-free
printf("%d\n", atomic_is_lock_free(&atomic_counter));

// Атомики всегда выровнены правильно
_Atomic(int) x;  // гарантированное выравнивание
```
